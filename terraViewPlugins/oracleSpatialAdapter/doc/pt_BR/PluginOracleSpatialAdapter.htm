Plugin Oracle Spatial Adapter

O Plugin Adaptador Oracle Spatial (Oracle Spatial Adapter) foi desenvolvido para adaptar tabelas espaciais do banco Oracle Spatial para o modelo conceitual da TerraLib.

1. Na primeira lista são carregados os nomes das tabelas espaciais do banco Oracle.

2. Selecione a tabela de interesse e clique em Adicionar.

3. Para alterar o nome da tabela a ser adaptada, clique em Adicionar Como e defina um novo nome para a tabela.

4. Para selecionar todas as tabelas, clique em Adicionar Todos.

5. Os nomes das tabelas selecionadas para adaptação são exibidos na segunda lista.

6. Para remover uma tabela da lista, selecione o nome da tabela e clique em Remover.

7. Para remover todos os nomes da lista de tabelas a serem adaptadas, clique em Remover Todos.

8. Após selecionar as tabelas a serem adaptadas, clique em Executar.

9. Ao final da adaptação será exibida uma mensagem de sucesso da operação e será criado um plano para cada tabela selecionada.
