Shapefile Theme Plugin

This plugin objective is to allow a TerraView user to create a theme from a file with vector data in Shapefile format, without importing this file into the database. This theme functionalities will be limited.

To create a shapefile theme, you must first open the main Plugin interface through the option File Themes of the Plugins menu, highlighted by the red circle shown in Figure 1. Other ways to do the same thing are by using the button in the main toolbar, also highlighted, or by selecting the option Add File Theme in the Theme menu.

Figure 1 Access to the Shapefile Theme Plugin.

Shapefile Theme Plugin main interface is shown in Figure 2.

Figure 2 Shapefile Theme Plugin main interface.

In this interface execute:

  1. Click on File button and choose a shapefile from your file system;

  2. If projection information is available, it will be presented in the appropriate field. Otherwise, click on Projection button to set it manually. If not set, the theme will be associated to "NoProjection";

  3. Choose the View in which the theme will be inserted;

  4. Choose a name for the new theme;

  5. Click on Ok button to finish the theme creation, or

  6. Click on Preview button to display the Plugin interface with some metadata and the drawing of the geographic component before creating the theme.

If Preview button is clicked, the interface shown in Figure 3 will be displayed.

Figure 3 – Verifying the contents of a shapefile.

To finish the theme creation click on Ok button. Note that the shapefile theme is presented in the TerraView Themes Tree (Figure 4) and that its geometries are displayed in the draw area.

Figure 4 - TerraView with a shapefile theme.

To visualize the attributes of objects in a shapefile select the information cursor () and click on the object. A table with its attributes will be displayed (see Figure 5).

Figure 5 – Attributes of objects of a shapefile theme.

A shapefile theme has limited functionalities, which are shown in the menu presented when right-clicking on the theme (Figure 6). The presentation and pointing default visual, rename and remove theme, define visible representations, and verify properties are the available functionalities (Figure 7).

Figure 6 - Shapefile Theme options.



Figure 7 - Shapefile Theme properties.

If a theme refers to a file and this file is deleted, the theme will be invalid, could not be visualized, and the menu displayed by right-clicking will present only the options to remove the theme and to display the theme properties.
