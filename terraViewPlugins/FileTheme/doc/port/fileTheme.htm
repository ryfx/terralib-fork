Temas de Arquivos Shapefile

O objetivo desse plugin é permitir que o usuário do TerraView crie um tema a partir de um arquivo de dados vetorias em formato Shapefile, sem que se faça a importação para o banco de dados. Esse tema terá funcionalidades limitadas.

Para criar um tema de arquivo faça acesse a interface principal do Plugin através do menu Plugin como mostra a Figura 1.

Figura 1 - Acesso ao Plugin de temas de arquivos.

A interface principal do Plugin de Temas de Arquivos é mostrada na Figura 2.

Figura 2 - Interface principal do Plugin de temas de arquivos.

Nessa interface faça:

  1. Clique no botão Arquivo e escolha um shapefile em seu sistema de arquivos;
  2. Caso exista a informação de projeção ela será descrita no campo. Caso não exista, você deve clicar no botão Projeção e informá-la manualmente. Caso não o faça, o Tema estará associado com uma projeção não geográfica ("NoProjection");
  3. Escolha a Vista na qual o Tema será inserido;
  4. Escolha um nome para o novo Tema;
  5. Clique no botão Ok para finalizar a criação do Tema, ou
  6. Clique no botão Visualizar para obter na interface do Plugin alguns metadados e o desenho da componente geográfica antes de criar o Tema

Caso você clique no botão Visualizar, você vai obter a interface mostrada na Figura 3.

Figura 3 - Verificação do conteúdo de um shapefile.

Para efetivar a criação do Tema clique no botão Ok. Observe que o Tema de shapefile aparece na árvore de Temas do TerraView (Figura 4), e suas geometrias são mostradas na área de desenho.

Figura 4 - TerraView com um Tema de shapefile.

Para visualizar os atributos dos objetos contidos no shapefile selecione o curso de informação ()e clique sobre o objeto. Uma tabela com seus atributos será mostrada (ver Figura 5).

Figura 5 - Atributos dos objetos do Tema de shapefile.

Um tema de arquivos tem funcionalidades limitadas, que são mostradas no meu obtido clicando-se com o botão direito sobre o Tema (Figura 6). Você pode alterar o visual de apresentação default ou de apontamento, renomear ou remover o tema, definir quais as representações estão visíveis, ou ainda verificar suas propriedades (Figura 7)

Figura 6 - Opções do Tema de Arquivos.


Figura 8 - Propriedades do Tema de Arquivos.

Se um Tema faz referência a um arquivo e esse é removido, o Tema ficará inválido, não podendo ser visualizado e o munu obtido com o botão direito mostrará apenas as opções de remover o Tema ou ver suas propriedades.
