WMS Client TerraView Plugin
User Guide

Author: Gilberto Ribeiro de Queiroz (gribeiro@dpi.inpe.br)
Date: 12/08/2007

WMS Client TerraView plugin is the tool that enables visualization of layers from WMS (Web Map Service) servers. This guide presents all functionalities of this plugin.
