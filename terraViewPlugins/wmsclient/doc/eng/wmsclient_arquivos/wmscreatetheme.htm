Creating a Theme from a WMS

<< Previous | First | Next >>

To visualize a map from a WMS server, open the Plugins menu, and select the option WMS Client, highlighted by the red circle in figure 1 below. Other ways to do the same thing are by using the button in the main toolbar, also highlighted, or by selecting the option Add WMS Theme in the Theme menu.

Figure 1 - TerraView main window and the access to the WMS plugin.

The window shown in figure 2 will be displayed. Use it to type in the WMS address from which the map is to be obtained or to choose a previously cataloged service from the list by clicking on the selected option.

Figure 2 WMS selection window

If the wished server is not available from the list, type in its address as shown in figure 3. After typing in its address, click on Next button to continue to the window shown in figure 4. If the server address is valid, it will be automatically registered and will be available for future connections.

Figure 3 Typing in a new server address

In this window (figure 4), select the layers to be visualized in TerraView. If the + symbol is displayed in the layer list, the item can be opened to access a more detailed list of the layers available from the server.

Figure 4 Layer selection window.

In this window (figure 5), select the layer to be visualized in TerraView. If the + symbol is displayed in the layer list, the item can be opened to access a more detailed list of the layers available from the server. In the figure 5 example, once the layers list is opened, the list shown in figure 6 will be displayed.

Figure 5 Layer selection window

Any number of layers can be chosen by selecting the desired layers. Note that when a layer is selected, a pre-visualization of its contents is displayed in the right panel (Layer Pre-visualization).

When selecting a layer, if the message shown in figure 6 is displayed, do not panic. Your plugin may not support all image formats provided by the server. In this case, choose another image format, such as PNG, to be used in future map pre-visualizations.

Figure 6 Image format not supported by the Plugin

After selecting the layers, click on Ok button to create the themes in TerraView. A window similar to the one shown in figure 7 will be displayed to confirm or not the themes creation. If the layers creation is confirmed, the plugin window will be closed, and the new themes (WMS) will be shown associated to the current view.

Figure 7 – Confirmation window for themes creation

WMS themes have a different icon with a "w" next to "T"letter. In figure 8 below, the two selected themes in the previous steps will be displayed in the Brasil view.

Figure 8 - Themes from WMS

Note that various themes can be created, from different services. Next, the query of information about a feature (geographic object) on a WMS map will be shown. Click here to learn how to query information about features on WMS maps or continue to the next page.
