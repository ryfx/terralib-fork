Querying Information About Features in WMS Themes

<< Previous | First | Next >>

To obtain information about any feature presented in a WMS theme, make sure that the WMS theme is active and that the theme is the current one (selected in the views tree). Next, click on the information cursor button (highlighted in figure 1 below) of TerraView tool bar,:

Figure 1 – Information Cursor

Move the mouse cursor to the feature with the desired information and click on the map. A window similar to the one shown in figure 2 will be displayed with information retrieved directly from the WMS server about the queried feature.

Figure 2 - Window with information about a feature

In the window presented in figure 2, the following information is displayed:

  • The name of the queried layer; in this example: Biomas do Brasil (Brazil Biomes);

  • A text box with attribute names and their values. In this example, they are gid, cd_legend, and id attributes with the following values, 3, CERRADO, and 3, respectively.

To close this window, click on Ok button.
