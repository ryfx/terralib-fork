Getting Information about a WMS Theme

<< Previous | First | Next >>

Information about WMS that provides a given theme can be obtained. To achieve that, right-click on the theme name. An option menu similar to the one shown in Figure 1 (below) will be presented. Select Properties... option and a window similar to the one shown in Figure 2 will be presented.

Figure 1 – Displaying details about a WMS

In the WMS theme window (Figure 2), the following information is displayed:

  • Service information : service title, service keywords, responsible contact information (Figure 2);

    Figure 2 - WMS theme properties window

  • Layer information : Projection and bounding box (Figure 3);

    Figure 3 - information sobre a layer

  • Configuration: parameters currently being used by TerraView plugin for queries to the server (Figure 4). Type of image format queried to the server can be changed, as well as the type of return format for pointing queries about features (Figure 4);

    Figure 4 – Configuration parameters for the communication with server

  • Styles: presentation styles of the layer made available by the server.

    Figure 5 - Presentation styles of a layer

If any configuration parameter is changed or if a different type of presentation style is chosen, the Apply button will be enabled (Figure 6) to save the changes. The following queries to the server, in draw and pointing queries, will be based on these changes and used to communicate with the server.

Figure 6 - Modifying the configuration of a theme
