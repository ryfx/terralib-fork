Visualizing Legends of WMS Themes

<< Previous | First | Next >>

To display the legend of a WMS theme, right-click on the theme name. An options menu similar to the one shown in figure 1 will be presented. If the WMS server has a legend associated to the theme, the Display Legend option will be enabled and can be selected. If the server does not have a legend, this option will be disabled and cannot be selected .

Figure 1 - Displaying a WMS theme menu: Display Legend

When Display Legend option is selected, the WMS plugin will retrieve the legend from the server and it will be available under the name of the theme on which the option was selected (figure 2).

Figure 2 - Legend Display

To hide a legend, click on the - symbol next to the theme name, or remove the legendpermanently from the views tree by choosing the Hide Legend option, as shown in figure 3 below:

Figure 3 - Hiding a legend

Figure 4 shows the result when the Hide Legend option is selected.

Figure 4 Hidden legend
