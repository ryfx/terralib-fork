Plugin Cópia de Banco de Dados

A figura abaixo mostra a terceira janela do Plugin de Cópia de Banco de Dados. Essa janela será exibida somente se a opção Copiar Planos da Informação na janela anterior for selecionada. Nessa janela deve-se selecionar os Planos de Informação que se queira copiar para o banco destino.

1. Na coluna Planos de Informação marque/desmarque os planos de informação desejados.

2. Clique em Selecionar todos ou Remover a seleção de todos se desejar, para facilitar a seleção dos planos de informação.

3. Clique no botão Próximo.
