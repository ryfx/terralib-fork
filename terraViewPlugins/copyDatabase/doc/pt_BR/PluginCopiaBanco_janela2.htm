Plugin Cópia de Banco de Dados

A figura abaixo mostra a segunda janela do Plugin de Cópia de Banco de Dados. Nessa janela deve-se informar o tipo de cópia, como descrito a seguir.

1. Na área Operação, marque a opção Copiar Banco de Dados para copiar todo o banco de dados, inclusive vistas e temas ou marque a opção Copiar Planos de Informação para selecionar na próxima janela os planos de informação a serem copiados.

2. Na área Opções marque a opção Criar as vistas do usuário corrente ou a opção Copiar as vistas de todos os usuários para os SGD'S que trabalham com usuário e senha. A opção Setar o Usuário das Vistas como o Usuário do Banco é habilitada somente para a opção Copiar as vistas do usuário corrente.

3. Clique no botão Próximo para prosseguir ou clique em Anterior para alterar os parâmetros da cópia.
