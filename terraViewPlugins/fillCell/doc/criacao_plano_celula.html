Criação do Plano com Representação Matricial

Anterior || Próximo || Página Inicial

  1. Selecione o plano de informação que será utilizado como base.
  2. Para criar um plano de células, utilize no TerraView a opção de menu Plano - Criar Células... ,ou, clique com o botão direito do mouse sobre o plano e escolha a opção Criar Células.
  3. Uma nova janela será apresentada.
  4. Observe nesta interface que as áreas Referência e Retângulo Envolvente já vêm preenchidas com os dados do plano de informação correspondente ao tema corrente da vista corrente. Se este não for desejado, é possível escolher o plano usando a lista fornecida, como também um outro tema (que poderia conter apenas um subconjunto dos objetos do seu plano).
  5. Preencha as resoluções para os eixos X e Y (que podem ser diferentes) na área Novo Plano de Informação.
  6. Entre com um nome para o plano de informação no campo Nome.
  7. Clique no botão Executar.
  8. Verifique o número de células, clique no botão Sim.
  9. Ao criar um plano de informação de células, o TerraView cria automaticamente uma tabela estática, com o mesmo nome do plano de informação.
