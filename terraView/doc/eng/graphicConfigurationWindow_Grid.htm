Graphic Grid Configuration

The interface for configuration of a graphic grid is shown in Figure 1.


Fig. 1 - Interface for configuration of a graphic grid.

The interface selects if grid lines are visible or not. Use Horizontal Lines Visibility and Vertical Lines Visibility fields to select the visible lines. Major box enables major lines and Minor box enables minor lines.

Background Color field selects the background color by clicking on Color... button by activating the color selection interface.

Color, style, and width of grid lines are set by MajorLines and Minor Lines fields.

Style list selects line style among:

  • NoPen,
  • SolidLine,
  • DashLine,
  • DotLine,
  • DashDotLine, and
  • DashDotDotLine
Width list selects line width from 1 to 4.
