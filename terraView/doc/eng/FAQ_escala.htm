How to change the Graphic Scale Unit in the Draw Area?


Right-click on the Draw Area to display the context menu, and select the Graphic Scale Unit option. Click on the unit of your choice from the list of available ones.

Note : If the projection of the view is "No Projection", physical units can not be used to represent the graphic scale, since there is no defined projection. In such case, the graphic scale will use "Units" as unit.
