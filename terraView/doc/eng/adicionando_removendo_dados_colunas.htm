Adding and Removing Data in a Column

It is also possible to create or to remove columns in a table.

1. Activate the popup menu from the POP2000 column header and select the Add Column option.

2. Define in which table (Nova_Tabela) the column will be created.

3. Define the data type of this column as Real. If String option, that corresponds to text, is selected, it is also possible to choose the column size.

4. Name the new column (den_pop).

5. Click OK button.

Insert data into column

1. Activate the pop-up menu from the den_pop column header, which has just been created and choose the Change Column Data option.

To calculate the population density in the year 2000 in each neighborhood the POP2000 column will be needed.

2. Define the operation that will result in the column new data. For this select a POP2000 column, the division operator ( / ) and the AREA column.

3. Click OK.

Note that the den_pop column was filled with the result of the operation defined in step 2.

Removing columns and tables

1. To remove a column, activate the popup menu of the column header to be removed and select Remove Column option.

2. To stop visualizing a table for a theme, activate the theme, select its popup menu, and choose Select Theme Tables option.

3. To remove tables created or associated to a layer from a database, activate the popup menu of Bairros layer and choose Remove Table option (Nova_Tabela).
