Skater Plot Graphic Configuration

The interface for configuration of a Skater Plot Graphic is shown in Figure 1.


Fig. 1 - Interface for configuration of a Skater Plot Graphic.

The interface contains a field to change the color and style that indicates the reduction of variance by number of groups.

Color... button activates the color selection interface.

Style list selects line style among:
  • NoPen,
  • SolidLine,
  • DashLine,
  • DotLine,
  • DashDotLine, and
  • DashDotDotLine
