Manipulating Grid Area

1. The popup menu presented below will be displayed when right-clicking on a table line label:

2. Promote > Lines Selected by Pointing option allows the visualization in the Grid area of the all pointed objects at the top of the table.

3. Scroll Grid > Next Pointed Object option scrolls the table until the next pointed object. Scroll to previous object is also available.
