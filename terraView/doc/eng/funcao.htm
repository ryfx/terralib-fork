Function

The Function option is available in Graphic Parameters window. This option must be used when more than one line of an attributes table is associated to the same geographic object. This condition happens when the selected column belongs to an external table or to a temporal table (dynamic attributes). In this case, only one value will be presented in the graphic, based on the result of one of available functions:

Function Result
AVG Arithmetic mean of series value
MIN Minimum value of the series
MAX Maximum value of the series
SUM Sum of the series values
COUNT Number of values in the series
