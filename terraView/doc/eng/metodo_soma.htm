Add Method

This operation adds themes to create a new layer. The source themes must have the same type of vector representation and only one geometric type: polygons, lines, points, or cells only.

A theme is selected in this operation as basis and the name and attributes type of the new table are defined from it. In addition, the new layer geometries projection is also defined from the basis theme.

Add Method allows two individual maps (or themes) to be combined in a new map. Here, two parts of Belo Horizonte city will be combined to create a map with the whole city.

1. Select graphic Representation Type;

2. Define Input Themes to be combined;

3. Define the fields of the tables (maps) to be preserved in the output;

4. Select Output Layer Name of the new map;

5. Execute the procedure.
