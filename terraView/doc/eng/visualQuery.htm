Query Visual

Query visual color is used to visually differentiate the result of a query in TerraView. The same color used by spatial query is used in attributes query.

Note that only the color is defined. Other drawing attributes, such as polygon filling style, line style, size and type of points are maintained. The visual highlight is applied only by color.

Visual color is also used in the grid to display the objects pointed in the map. The corresponding lines are drawn with this color.

Query visual color is defined by the color selection interface.

Another method to display the result of a query is to show, in the grid and on the map, only the objects that satisfy the query. This method is easy to implement and quicker to display the result, but is not used by TerraView because the context of the objects is lost.

The method selected by TerraView maintains the context and makes the analysis of the objects and their surroundings better.
