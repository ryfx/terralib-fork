Calculating Area and Perimeter

1. Activate cell popup menu and choose Calculate Area option. This option is available only if the theme contains polygons. TerraView will calculate automatically the area of each neighborhood and store it in AREA column, which is also automatically created in the table of the original theme. Observe that if a column with this name already exists, the option will not be available.


2. Select Calculate Perimeter in the popup menu to calculate polygons perimeter. The program will store the perimeter values in PERIMETRO column, which is also created automatically like the area case.
