Presentation Format


Observe that TerraView data are always presented in a tree format.

In Databases tree, the opened databases are presented and in the second level, all database layers are presented.

In Views tree, Themes associated to the view of the active database are presented.

An active database is the selected one and, by double clicking on its name in the Databases tree, it is highlighted in blue. The active view is the selected one, highlighted also in blue, and the theme related to it is visualized in the draw area. To activate a view, double click on a theme, and it can be visualized using Draw operation, with button in the tool bar.

Layer, View and Theme can be renamed as needed. To rename, select the Layer, View or Theme, right click to obtain the popup menu and select Rename option.

All Themes can be put in a single view to visualize them simultaneously. To execute this operation, drag the desired themes to the new view, or add them to the view. In the example below, all available graphic information about São Paulo districts are visualized in BH view.

Note: Observe that in the view window the active themes for visualization are easily known.
