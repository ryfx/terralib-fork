Building Histograms

Open BH.mdb database. Deactivate all themes of BH view, but SetoresCensitarios. With this theme selected, click Draw button. Observe that the map polygons are transparent and will not be possible to visualize pointed oobjects. To change the transparency, use the popup menu of SetoresCensitarios theme and select Visual > Default; choose Style as Solid and click OK.

In grid area at the button of the interface, notice that there is a column called REN_MED, with information about the average monthly income of household heads at each census tract of Belo Horizonte. To obtain the distribution of the average income in the census tracts of Belo Horizonte city, execute the following steps:

1. With SetoresCensitarios theme activated, use the popup menu and choose Graphic Parameters.

2. In the Graphic Parameters that is displayed choose Graphic Type as Histogram to obtain the distribution of the attribute to be selected.

3. Choose the REN_MED column as Attribute(X) (a shortcut may be used by selecting Attribute(X) field and pressing the initial letter R).

4. In Slices Number field, choose how many intervals the graphic will be broken into. Use default option 30.

5. In Objects field define if the histogram will be executed with all objects, pointed ones only, queried ones only, and other modes. Use default option All to create a histogram with data from Belo Horizonte census tracts. Click OK.
