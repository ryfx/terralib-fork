Difference operation between themes

Difference operation is executed between themes with geometric representation of polygons type. It is used to remove whole objects or parts of its geometry. In the following example, input theme Tema1 representing Brazil municipalities and theme Tema2 with clip area are operated. The difference operation removes all municipalities contained in Tema2ellipsoidal region. Municipalities partially contained are not removed, but their geometries are clipped by the Tema2 ellipses. Figure 1 shows Tema2 overlaid on Tema1, before the difference operation.


Figure 1 - Theme with Brazil municipalities to be clipped by Tema2.

Figure 2 shows the result of the difference operation.

This operation does not change any themes. The result is the creation of a new layer.

Difference between themes interface

The difference between themes interface is shown in Figure 5 and to activate it, use the context menu on a View, selecting Geoprocessing - Difference. In Theme 1 (Original Theme) box, select the theme to be clipped.


Figure 5 – Difference between themes interface.

Use object selection box of original theme to set the objects to be clipped: all, pointed only, or queried only.

In Theme 2 (Theme to be subtracted from the original) box, select the theme to be used as overlay mask. Use object selection box to set the objects to be clipped: all, pointed only, or queried only.

In Output Layer Name box, type the name of the output layer to be created.

OK button applies the difference operation.

Cancel button disables the interface.

Help button activates the help interface.
