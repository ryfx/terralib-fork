Raster Histogram Graphic Configuration

The interface for configuration of a Raster Histogram Graphic is shown in Figure 1.


Fig. 1 - Interface for configuration of a Raster Histogram Graphic.

Color... button of "Brush" field activates the color selection interface to change the filling color of the histogram.

Color... button of "Line" field activates the color selection interface to change the outline line color of the histogram.

Style list of Line field selects outline line style among:
  • NoPen,
  • SolidLine,
  • DashLine,
  • DotLine,
  • DashDotLine, and
  • DashDotDotLine
