Default Visual

This is the presentation visual of theme objects that are not pointed. Assume that a theme contains areal or polygonal geometric representations. For this type of geometric representation there are two kinds of independent parameters. The ones related to polygon area: color, style and transparency percentage; and the ones related to polygon contour: color, width and style.
Changing the transparency of polygon fill is useful to avoid this polygon to overlay other themes drawn below it.
When the Default option is selected, a window will be presented.


Changing the Default Visual of a theme

1. Click on the Color button and the window with the color palette will be displayed to select the new color.
2. After choosing the color, click on OK button to return to the main window and click on OK button of that window. To check the result, redraw the theme
