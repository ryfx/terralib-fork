Active View / Active Theme

 

A database in TerraView can have various Layers. Different Views can be created for them as required, as well as different Themes can be created for their Views. A View must be active in order to their Themes be shown in the Draw Area using the shortcut. This allows the presentation of information from different regions and spatial levels. For instance, a database may contain data about Belo Horizonte city, the municipalities that are part of the Belo Horizonte metropolitan region, and all municipalities of Minas Gerais state, but they can be visualized separately.
