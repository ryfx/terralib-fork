Spatial Query with Only One Theme

A very interesting operation is to associate the objects of a theme according to their geometries, in a similar way to spatial pointing. It is possible to determine neighborhood relationships and crossings, making analysis much easier. Furthermore, it is possible to create a new layer with selected data if desired.

Polygons

Suppose the following situation: In order to investigate the region where Bandeirantes neighborhood and its neighbors are, a layer in TerraView that contains only this region can be created:

1. Set Bairros theme active and point only Bandeirantes neighborhood. (Use column sort to facilitate this task and remove pointed colors and queries).

2. Activate popup menu from Bairros theme and select Spatial Query.

3. In Topological Relations choose Touch to find all the neighborhoods that border with Bandeirantes neighborhood.

4. Execute the operation and note that the neighborhoods that border Bandeirantes is now drawn with the color of queried objects.

5. Next, create a new layer called reg_bandeirantes from the theme selecting pointed objects (Bandeirantes) and queried ones (its neighbors). Use Create Layer from Theme tool in the Bairros theme popup menu. In Selection of Objects mark Queried or Pointed.

6. Create a new theme reg_bandeirantes from this layer in BH view. Note that only the neighborhoods of interest are going to be displayed.

Other available options to associate polygons are:

Operation Function
Disjoint No intersection
Equal Exact equal
Touch Borders with
Overlap Has intersection but is not contained
Cover Contained and touches border
Covered by Contains and touches border
Within Is contained but does not touch border
Contains Contains but does not touch border

Lines

Suppose that there is a task of defining all the streets or avenues that cross Afonso Pena Avenue. Because the data are stored by street segments it is necessary to select all the segments that compose this avenue.

1. Activate RuasCentrais theme, remove pointed and queried objects, and select all segments that compose Afonso Pena avenue (use column sort to easy this task).

2. Activate the popup menu and select Spatial Query.

3. In Topological Relations field, select Touches. Create as a New Query.

4. Observe that all lines that touch any segment of the avenue are selected. Execute a new spatial query by selecting Disjoint as topological relation and choose Add. Query options are similar to the attributes query: New Query, Add and Filter.

All central streets are now selected. The other topological relations available for lines are:

Operation Function
Disjoint No intersection
Equal Exact equal
Touch Ends at the line
Overlap Has a common segment
Crosses Has a common point (not at end point)
Within Is contained
Contains Contains
