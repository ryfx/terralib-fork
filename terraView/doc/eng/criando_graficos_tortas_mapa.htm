Creating Pie Chart on the Map

Homicides in the years 2000 and 2001 can be analyzed by another approach: by checking the proportion between the total of homicides in the two years for each census tracts. A pie chart is used for this type of analysis.

Create a new column called Total00_01 in SetoresCensitarios theme by using the popup menu from the column header in the Grid area, selecting Add Column option, and choosing Setores table.

Once the column is created, it must be filled in with the sum of homicides in the years 2000 and 2001 provided by HOM_2000 and HOM_2001 columns, respectively. To fill in, right click on the column label and select Change Column Data.

Then create a thematic map with Equal Steps by using Setores.Total00_01 attribute to color municipalities based on homicides total in the years 2000 and 2001. (Select 5 intervals).

Finally, to create a legend by Pie Charts, which allows the analysis of the homicides proportion in each year, execute the following steps:

1. Activate the popup menu of SetoresCensitarios theme.

2. Select Edit Bar or Pie Chart.

3. In Type field, select Pie Chart.

4. In Attribute field, select Setores.HOM_2000.

5. Click Insert. Observe that the attribute is represented in the grid.

6. In Attribute field again, select Setores.HOM_2001, and click Insert.

7. Select Fixed Size to keep the graphic size when the map scale is changed.

8. Click OK button.

Observe that the map, in addition to being colored by the total number of homicides, presents the proportion between the years 2000 and 2001 for each municipality.
