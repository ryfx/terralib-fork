Selecting Map Parameters

To create a map based on the minimum and the maximum number of homicides with six equal size intervals with a different color associated to each interval, execute the following steps.

1. In Mode field, select Equal Steps to create same size intervals. Other attribute intervals division types are:

  Type   Description
 Equal Steps  All groups have the same interval.  
 Quantil  All groups have approximately the same number of elements.  
 Standard Deviation  All groups have the same interval, calculated as a multiple of 1/4, 1/2 or 1 standard deviation.  
 Unique Value  Each group corresponds to a distinct value of the attribute.  

2. In Slices field, select option 6 to color the map with six different colors. This option is selectable for Equal Steps and Quantil types. For Unique Value type, the number of intervals is defined automatically and, for Standard Deviation type, intervals are defined as 0.25, 0.5, or 1 of the standard deviation.

3. In Attribute field, select Bairros.POP2000 option. This field is used to select the attribute used to color the map.

4. In Precision field, select option 1. This field defines the number of significant zeros to be used.

5. Click Apply to create the groups. However the map is not colored yet.
