Other Spatial Selection Operations

In default mode, selected objects are those that intersect (are contained or touch) the selection cursor. In addition to default Intersects option, TerraView allows the user to choose other spatial operation types with the cursor.

Spatial Operation options do not affect objects equally, since they have specific effects for each type of geometry, as shown in the table below:

Operation Points Lines Polygons
 Intersects   Inside the cursor   Inside or crosses the cursor   Inside or crosses the cursor 
 Disjoint   Outside the cursor   Outside the cursor   Outside the cursor 
 Within   Inside the cursor   Inside the cursor   Inside the cursor 
 Crosses   No   Crosses the cursor   Crosses the cursor 

Spatial Operation option Within selects all areas fully inside (are contained) the cursor.

If the user chooses Spatial Operation option No Spatial Operation, the cursor will not select areas.

To unselect areas selected by the Graphic Cursor, click outside the map or use the tool to unselect objects.
