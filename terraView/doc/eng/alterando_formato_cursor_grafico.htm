Changing the Shape and the Type of Graphic Cursor


Changing the Graphic Cursor Shape

To increase or decrease the size of the rectangle press Alt key and move the mouse until desired size is obtained.

The triangle orientation can be changed by pressing Ctrl key and moving the mouse.

To execute the two operations altogether press AltGr key.

Note: The Alt and Ctrl keys (pressed together) also have the same function as AltGr key.

Changing the Graphic Cursor Type

1. Activate the popup menu in the Draw Area.

2. Select the option Spatial Cursor Type > Ellipse. This option will only be available if the Graphic Cursor is activated.

The selected cursor becomes an ellipse and its size and orientation can be changed as previously described.
