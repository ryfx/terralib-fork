Create Layer from Theme

By creating layer from a theme it is possible to make copies of whole layers or parts of layers using theme restrictions. It is also possible to change the projection of a layer using this procedure and saving it in a new layer.
To create a layer from a theme, right-click on the layer name and select Create Layer from Theme option, as shown in the following figure.




Define the name of the new layer and choose the desired projection, which can be the View projection, the Layer projection, or a different one by selecting User Defined. To finish, click on OK.
