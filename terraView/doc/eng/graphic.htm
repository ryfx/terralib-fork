Graphic Plot

Graphic plot is executed in the interface shown Figure 1. TerraView uses QWT library for graphic plot.

Types of graphics plotted by TerraView are:

  • histogram,
  • raster histogram,
  • dispersion,
  • diagrama de espalhamento de Moran e
  • normal probability.
Brushing

Brushing is selected by activating Spatial Selection mode. The selection is executed through the tool bar in the TerraView main interface. Figure 2 shows the tool bar and the activation button is pointed by a red arrow.


Fig. 2 - TerraView tool bar with the button to activate the "Spatial Selection" mode.

Brushing operates on histogram and dispersion graphics to select objects through cursor. In histogram graphic the brushing cursor has the shape shown in Figure 3 and selects objects with values between xMin and xMax.


Fig. 3 - Brushing cursor for histogram graphics.

In dispersion graphic the brushing cursor has the rectangular shape shown in Figure 4 and selects objects with values within xMin, xMax and yMin, yMax.


Fig. 4 – Brushing cursor dispersion graphics.

Objects are selected by the mouse left button.

Selected objects are drawn (highlighted) on the 3 data presentation modules:
  • map display,
  • attribute table, and
  • graphics window.
Keys to control the brushing cursor
  • "ALT" key resizes the cursor.
  • "SHIFT" key adds the new selection to the already selected set.
Tool Bar

Tool bar is shown in Figure 5.


Fig. 5 – Graphic display tool bar.

The tool bar contains toggle and push buttons. Toggle buttons are used to select an operation mode and push buttons are used to execute a task.

Toggle buttons are:
  • "Promote" and
  • "Zoom".
These buttons are highlighted to show that they are activated. Figure 6 shows activated "Promote" and "Zoom" buttons, indicating that promote and zoom operations are selected.


Fig. 6 - Graphic display tool bar with activated "Promote" and "Zoom" buttons.

"Promote" operate the attributes table module and, when active, selects the automatic promote mode of objects pointed by the mouse (selected).

"Zoom" button, when active, allows the cursor on graphics window to execute zoom. Zoom has higher priority than brushing; therefore, to return to brushing mode, the "Zoom" button must be deactivated.

Pan on graphic window

"Zoom" button must be active to execute pan operation. Pan is controlled by the arrow keys.

Push button are:
  • "Previous",
  • "Next",
  • "Config",
  • "Print", and
  • "Help".
Previous" and "Next" buttons are enabled when zoom is active. They are used to forward and backward zooms and pans executed in the graphics window.

"Config" button activates the configuration interface to change the visualization of:
  • histograma
  • raster histograma
  • dispersion
  • normal probability
  • grid
  • font of x and y axis
"Print" button activates the print window.

"Help" button shows this help window.
