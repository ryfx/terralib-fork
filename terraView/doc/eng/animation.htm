Animation

This tool animates the results of a temporal operation or of a simulation.
It is possible to animate a sequence of images with speed control, pause, and step-by-step movement.
A folder is opened by the tool and its contents are animated sequentially by alphanumeric order of images names.
Some accepted file formats are: jpeg, png, and bmp.

The user must create a folder and insert the sequence images.

Right-click on the Draw Area and select Animation, as shown in the following figure.



Click on Directory button and select the folder where the images are stored.




Click on Play button to display the images.


Application example: If there is a temporal data with dairy production in Vale do Paraíba farms, a grouping by month of this data can be built and animated to visualize the dynamics of dairy production in these farms.
