Creating a Table

New attributes can be inserted into objects. For instance, neighborhood areas can be calculated or a new column with population density can be created.

1. Activate the popup menu of Bairros layer and select Create Layer Table option. Create Table window will be displayed.

2. Define the name of the new table (Nova_Tabela). A new table is associated to Bairros layer and is available to store new information.

3. To associate the table to the theme, activate the popup menu of Bairros theme and choose Select Theme Tables. Insert Nova_Tabela table in Bairros theme. Notice that only one new column was added.


Next insert data in the theme tables.

To execute the operations below activate Bairros theme by highlighting it in blue.
