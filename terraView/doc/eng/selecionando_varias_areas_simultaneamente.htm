Selecting Many Areas Simultaneously

It is possible to select objects not by their attributes, but directly from the map. In order to allow this type of selection, TerraView provides the Pointer Cursor to select one object at a time, and the Graphic Cursor .

Activate Graphic Cursor to select one or more objects simultaneously. The default option is a rectangular shaped cursor which selects all the objects intersected by the rectangle, as shown below.
