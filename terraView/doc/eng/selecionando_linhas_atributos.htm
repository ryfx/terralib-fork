Selecting Lines and Attributes

Every graphic object (maps) has an associated attributes table with information about this object. For instance, neighborhoods of Belo Horizonte city have associated information about code (NUBAIPOP), name (NOBAIPOP), region name (TIPBAIPOP), and population in the year 2000 (POP2000). Observe that there are "n" lines in the Grid area and that they correspond to the "n" neighborhoods that are visualized in the Draw area. In this way, when lines of the grid are clicked or selected, the neighborhoods connected to these lines will be highlighted in the Draw area.

Select one line of the table using the Pointer Cursor .

Click on the chosen line label to select that line. Observe that the region corresponding to the line will be displayed with a new color and that this color will be the same to highlight the selected line.

 

At the bottom of the interface some information about the table is presented, such as the number of lines and the number of pointed lines. When pointer cursor is used to select a region, the region and the corresponding line(s) in the grid will be displayed in different colors and the grid will display automatically the line that corresponds to the object.

To always display the selected region in Draw area, activate the popup menu (right click) on the Draw area and select Centralize the Drawing of the Selected Objects. If the region is not visible, a redrawing will be executed with the region in the center. Otherwise, the region will be highlighted as before.
