Exporting Data

Up to this point, importing of data by various means into TerraView has been presented. Data can also be exported in vector format, that is, with geographic information using vector format and its associated attributes. Available export formats are SPRING, MID/MIF, Shapefile, and TXT.

1. Select the layer to export. Select UP.

2. Select the layer attributes table to export. In a similar way to previous external table import, verify that there are two tables to export, choose UP table only.

3. Select the file format to be created. Select MID/MIF.

4. Define the name of exported file. Choose Data directory and UP_2000 file.

5. Apply the procedure.
