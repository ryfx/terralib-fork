Creating layers from themes query

A layer can be created from a query executed on a theme. The new layer can be used to create different themes, avoiding the need to re-execute the query. For instance, to analyze only slums (“favelas”) with area over 9875 square meters, execute the following steps:

1. Activate Favelas theme, select the popup menu of a cell, and choose Calculate Area option.

2. Check that there are no queries from Favelas theme (if there is, execute the procedure to remove the visualization of queried objects).

3. Query the desired objects (AREA > 9875).

4. Activate the popup menu of the theme and select Create Layer From Theme option.

Create Layer From Theme window will be displayed and use it to define:

1. Selection of Objects type to specify the objects of the new Layer. Select Queried.

2. Name of the new Layer: Favelas_1.

3. Select View. The projection of the layer must be set.

After executing the creation, the new layer will be shown in Databases tree and can be used to create Themes.

Create a new Theme Favelas_1 from the new Layer by clicking on button and add it to the View the theme should belong to. The visualization of slums with area over 9875 square meters is now possible.

Creation of layers from themes can also be used to re-project data and to create themes from pointed objects.
