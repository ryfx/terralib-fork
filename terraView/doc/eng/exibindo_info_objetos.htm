Displaying Objects Information

In TerraView, a window with detailed information about an object can be displayed. To obtain it, execute the following steps:

1. Select the object by pointing.

2. Go to the corresponding line in the grid.

3. Activate the Line popup menu and choose Attributes and Media option. The program will display a table with all information about that line.

The same information can be displayed in the Draw area by:

1. Selecting the Information Cursor

2. Clicking on the desired object.

3. Information about the selected area will be displayed.
