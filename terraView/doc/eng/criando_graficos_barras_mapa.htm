Creating Bar Charts on the Map

Edit Legend allows also the association of graphics to map regions to show some of their attributes. Available graphics are the bar charts and the pie charts, in which different attributes can be visualized simultaneously.

Bar charts are suitable to analyze the difference in absolute numbers, while pie charts allow analysis of proportions in each area.

Open BH.mdb database. Deactivate all themes of BH view, but SetoresCensitarios. With this theme selected, click Draw button. If the map polygons are transparent, use the popup menu of SetoresCensitarios theme and select Visual > Default; choose Style as Solid and click OK.

To analyze the evolution of the number of crimes in each census tracts of Belo Horizonte in the years 2000 and 2001, execute the following steps.

Activate SetoresCensitarios theme

1. Activate the popup menu of SetoresCensitarios theme.

2. Select Edit Bar or Pie Chart.

3. In Type field, select Bar Chart.

4. It is not necessary to change Height and Width fields.

5. In Attribute field, select Setores.HOM_2000.

6. Click Insert. Observe that the attribute is presented in the grid.

7. In Attribute field again, select Setores.HOM_2001, and click Insert.

8. Click OK butoon.

Now, for each census tract of Belo Horizonte there are two bars representing the homicides in the years 2000 and 2001. Observe that the graphic parameters were inserted in the Views tree under the SetoresCensitarios theme.

Modifying a legend

There are two options to modify a legend:

1. Activate the popup menu of SetoresCensitários theme and select Edit Bar or Pie Chart option.

2. Activate the popup menu of legend (Bar Chart) shown with the SetoresCensitarios theme and select Edit Bar or Pie Chart.

By selecting any of the option above the Edit Bar or Pie Chart window will be displayed.

Modifying legend labels

The displayed labels are the names with the colors of the columns selected as attributes. To modify labels, execute:

4. Right click on a label (bar) to modify it.

Name and color of bars can also be modified by:

5. Right click on the bar to modify it.

6. Select Set Alias option.

7. Define a new name for the bar and press Enter key.

8. To retrieve a previous name, select Remove Alias option.

9. To modify the color of a group, select Change Color option.

It is not possible to have more than one graphic on the same theme and the creation of a new graphic will overwrite the previous one. However it is possible to have a thematic map and a graphic for the same theme.

Removing graphic

Activate the popup menu of Bar Chart legend shown with SetoresCensitarios theme and select Remove Chart option.
