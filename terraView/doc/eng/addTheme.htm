Creation of Local Themes

A theme is a subset of objects from a layer, and may contain all objects of that layer. Every theme is under a view, which has a specific cartographic projection.

A theme is called local theme or just theme when its layer belongs to an active database, that is, the one TerraLib client application is connected to.

To create a local theme, execute the following steps:

1. Click on Theme - Add menu, as shown in figure below, or click on the icon of the tools bar.


Creation of local theme

2. Add Theme interface will be displayed. Choose the layer in Infolayer, select the view where the theme will be visualized in View, define the name of the theme to be created in Theme Name, and click on OK button, as shown in the figure below.
