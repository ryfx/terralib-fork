Dispersion Graphic Configuration

The interface for configuration of a Dispersion Graphic is shown in Figure 1. Dispersion is drawn with the colors defined for the theme. Pointed (selected) objects by the mouse are drawn with the pointing color of the theme and the not pointed objects are drawn with the default color of the theme.


Fig. 1 - Interface for configuration of a Dispersion Graphic.

"Style" list selects symbols among:
  • Ellipse,
  • Rect,
  • Diamond,
  • Triangle,
  • DTriangle,
  • UTriangle,
  • LTriangle,
  • RTriangle,
  • Cross, and
  • XCross
"Size" list sets the symbol size from 1 to 15.
