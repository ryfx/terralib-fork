Composed Query

It is possible to execute queries composed by many selection clauses (attribute, operator, value). For instance, it may be necessary to select neighborhoods with area over 651628.2 square meters (if needed, create the area column by activating the popup menu of a cell and selecting Calculate Area option) and population in the year 2000 larger than 6000 inhabitants. A single query can be executed using AND, OR, and NOT operators, and parentheses to compose the query. Or the queries can be executed in an incremental way using Add and Filter buttons. A new clause followed by Add button is equivalent to the use of OR operator: New objects from the query are added to the already selected objects. Filter button is equivalent to AND operator: all objects that do not belong to the query are removed from the already selected objects.

1. In Bairros theme, select neighborhoods with area over 651628.2 square meters and population larger than 6000 people.
