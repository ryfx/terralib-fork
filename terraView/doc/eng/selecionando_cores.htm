Selecting Colors

Displayed colors can be modified. New color mixture is created using color options provided by the program using the buttons in the lower left part of Legend Editor. Each button has a letter corresponding to its color: R for red, G for green, and B for blue. When selecting Gradual Color Mixture, the transition from one color to another is smooth. Variation of colors from a light to stronger color or vice-versa can be selected, as well as restarting color selection.
