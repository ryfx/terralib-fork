Export Table As

This tool allows saving lines and columns of a grid in a text file.

Selection: In Lines field select the lines to export: all lines, pointed ones only, queried ones only, or visible lines only. In a similar way, Columns field is used to select the columns to export: all columns, visible ones only, or selected ones only.





Delimited With: in this box select the separator character of the text file to be created either by choosing one of the available characters in the interface or by choosing Other and typing in the character you wish to be used as a separator.

Click on Browse and select the folder and the File name to be created.

To export click on Ok button.
