Normal Probability Graphic Configuration

The interface for configuration of a Normal Probability Graphic is shown in Figure 1.


Fig. 1 - Interface for configuration of a Normal Probability Graphic.

The interface contains a field to change the color and style of the normal curve and another field to change the color and style of the normal probability curve.

Color... button activates the color selection interface.

Style list selects line style among:
  • NoPen,
  • SolidLine,
  • DashLine,
  • DotLine,
  • DashDotLine, and
  • DashDotDotLine
