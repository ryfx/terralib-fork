Using Brush Tool

Use tool bar buttons to define which information to visualize. The set of buttons activate and deactivate the display of Database Tree, View Tree, Grid, Draw Area, and Graph Window, respectively. Select them in such way that the trees are not displayed. Next, select Tile Windows button to resize the Draw area and Graph windows and let them use the whole area of the system main window.

To verify where the census tracts with mean monthly income below R$ 500.00 are, execute the following procedure. Select Graphic Cursor to select data in the histogram. To change cursor width, press Alt key and move the mouse. Click on graph X axis (REN_MED) to select the range of mean income to display on the map. Select the range from R$ 0.00 to R$ 500.00.

Observe that the bars of sectors with mean income between zero and five hundred reais are selected in the histogram (in green); in the map, the sectors related to this selection are also highlighted in green, as well as in the grid, where the records corresponding to the objects are selected (use Promote > Lines Selected by Pointing in lines popup menu).

Select a census tract on the map to verify the relative position of that sector in the histogram. This type of combined browsing is known as brush and facilitates data analysis. Remove the pointing selection using SetoresCensitarios theme popup menu and choose Remove Colors > Pointed Objects
