Deleting Columns, Layers, and Themes

1. To delete created Area column, activate popup menu on AREA column and select Delete Column option.

2. Activate Bairros1 theme, use the popup menu to select Remove option.

3. To delete Favelas_1 layer, activate popup menu and select Remove option.
