Creating Database

Select database option from shortcut or from File > Database menu:

1. Select Create option.

2. Define the type of database to create. TerraView can use different DBMS, such as Access, Oracle, and MySQL. Select Access option.

3. Set Bancos directory as the location where the new database will be saved.

4. Set a name for the database.

5. Click OK to apply the procedure.

If the selected manager is ACCESS, the database will be a file with ".mdb" extension in the selected directory. For other managers, the database will be created in the selected server (host). In this database, geographic data related to an application are stored.
