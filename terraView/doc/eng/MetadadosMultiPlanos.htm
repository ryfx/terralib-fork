Multiple Layer Metadata

Information about Layers, such as Author, Source, Quality, Description, Date, Time, etc, can be recorded. These additional information about Layers are called metadata. When Properties is clicked in the context menu of a Layer, in the Databases Tree, a window is displayed with information about the Layer, as shown in the figure below. The highlighted fields are (items 1 to 8) metadata.

The user can change the metadata information directly in this window or change metadata for several layers at once using the tool available from menu item Layer - Update Metadata, shown in the figure below.

1. In Input Layers list, select the layers with metadata to change and click on to add to Selected Layers list, and to remove a selected layer, select it in the Selected Layers list and click on .

2. To add all layers to the Selected Layers list, click on and to remove all selected layers, click on .

3. If you wish, select in Selected Layers list, a reference layer to be used as the source of metadata information to be copied to other selected layers, as shown in the previous figure (AreasVerdes layer).

4. Click on Next. The following window will be displayed.

5. Click on Previous if you want to change the layers selection.

6. If a layer has been selected as the reference layer in Selected Layers field, its metadata will be shown in this window. Select All option to update all metadata or check only some metadata to update, as shown in the previous figure (Author, Source, Quality fields). If no layer was seleted as reference for metadata, type in the information directly in the fields to update (do not forget to check this fields for changing).

7. Click on Execute.
