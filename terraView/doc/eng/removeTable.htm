Remove Table


TerraView allows removal of external and layer tables. After a external table is removed it will not be possible to connect to the removed table and also, if a layer table is removed, it will not be possible to create themes using this table.

To remove an external table, click on Remove External Table from the context menu available when right-clicking on the database name in the Databases Tree. To remove a layer table, click on Remove Layer Table from the context menu available when right-clicking on the layer name in the Databases Tree. See below the interfaces used to remove a table.

Remove External Table

Remove Layer Table

  1. Select the table to be removed.
  2. Click on OK.
  3. The user will be warned if a view or a theme will be affected by the table removal. If you are sure on the removal, click on Yes.
