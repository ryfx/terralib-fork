Spatial Query with Two Themes

Queries that associate objects from two different themes are the most interesting in TerraView. It is possible to define, for example, the crimes committed in a neighborhood, the census tract of the neighborhood or the streets that connect two neighborhoods. For this, the spatial query must have an active theme, where the queried objects are selected and another theme where the objects to be associated with are.

Associating Polygons and Points

To select all the crimes committed in Centro and Savassi neighborhoods (called Savassi region) in 2003:

1. Set Bairros and Crimes2003 themes as visible and activate Bairros theme. Select Centro and Savassi neighborhoods.

2. Activate popup menu from Bairros theme and choose Spatial Query option.

3. Set the Theme option as visible and choose Crimes2003 theme. This selects the theme where the queried crimes are.

4. Mark Contain option in Topological Relations field. The lower part of the spatial query screen will display all selected items in the query. In this particular case, the selected items are all the crimes committed in 2003, in the two neighborhoods selected in step 1.

5. Press New Query button.

Note: In case Disjoint option was chosen all crimes that do not intersect with the selected neighborhoods would be selected. Touch option would select all crimes that were committed exactly in the neighborhoods borders.

Other relationship types

Depending on the object types, the active theme and the visible theme selected, the available spatial operations change. Bellow there is a summary of possible combinations:

Active Visible Operations
Polygons Polygons The same previously presented for operations with polygons only
Polygons Line Disjoint, Overlap, Touch, Crosses, Within, Covered by
Line Line The same previously presented for operations with lines only
Line Point Disjoint, Within, Touch
