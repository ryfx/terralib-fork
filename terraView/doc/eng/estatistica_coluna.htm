Columns Statistics

Statistical information about a column values are provided by TerraView and can help the data analysis.

1. Activate SetoresCensitários theme and select the REN_MED column.

2. Activate the popup menu of REM_MED column header and choose Statistic option. This option is available only for columns with numeric data. A new window will be displayed.

3. In Objects field, select the table lines to be used in the calculation. The default option is to use all lines.

4. In Summary field, select how lines will be grouped in statistics calculation. Select HOM_0003 option. Then the mean income statistics will be calculated based on census tracts grouped by the number of homicides in the sector.

5. Displayed data can be saved in text file (.txt) by selecting Save option.
