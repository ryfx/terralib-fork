Semivariogram Graphic Configuration

The interface for configuration of a Semivariogram is shown in Figure 1.


Fig. 1 - Interface for configuration of a Semivariogram.

The interface contains a field to change the color and style of the Semivariogram and another field to change the color and style of the Adjust Curve.

Color... button activates the color selection interface.

Style list selects line style among:
  • NoPen,
  • SolidLine,
  • DashLine,
  • DotLine,
  • DashDotLine, and
  • DashDotDotLine
