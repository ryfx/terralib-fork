Creating Themes with Objects Subset

A theme may be created with some objects of the visualized layer. For instance, it may be necessary to only analyze neighborhoods with a population larger than 9980 people in the year 2000. To allow this type of analysis the attributes selection tool is available during the creation of a theme.

1. Click button to add Bairros1 theme to BH view from Bairros layer.

2. Unset Select All of the Object option. An interface similar to attributes query one is displayed to enable filtering of objects to be in the theme.

3. Select only neighborhoods with a population larger than 9980 people in the year 2000.

4. In Theme Name field type in: Bairros1.

5. Create the new theme.

Notice that only neighborhoods with a population larger than 9980 people will be drawn on the map and in the attributes table.
