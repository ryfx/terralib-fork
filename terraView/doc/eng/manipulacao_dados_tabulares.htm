Sorting, hiding and swapping Columns

Open BH.mdb database and activate Bairros Theme.

Sorting columns

1. Highlight NOBAIPOP column by clicking on its name (This column will be selected). Records (table other columns) will be sorted in accordance with the selected column.

2. Right-click on the lower portion of the selected column to activate the cell popup menu. Choose one of the options to sort: Sort in Ascendant Order - to sort from smaller to larger, and Sort in Descendent Order - to sort from larger to smaller.

Hiding and displaying table columns

1. Select NUBAIPOP, NOBAIPOP, TPBAIPOP, POP2000, and object_id_1 columns by pressing Shift key and clicking the first and the last columns. All columns between these ones will be automatically selected. Multiple columns can also be selected using Crtl key with the difference of not selecting columns between clicks.


2. Activate the cell popup menu and choose Hide Column(s) option. All selected columns will be hidden.

3. Activate the cell popup menu and choose Show All Column(s) option to display all columns again.

Swapping two columns

1. Select NOBAIPOP column and select POP2000e column with Ctrl key pressed.

2. Activate the cell popup menu and choose Swap Columns option.
