Local Empirical Bayes

The Local Empirical Bayes estimator includes spatial effects, calculates the rate locally by using only the geographic neighbors of the area where the rate is to be estimated, and converges to a local mean. Corrected rates are less unstable since their calculation considers not only area information, but also neighborhood information. If a location has a sizable population, the rate will have small variability and will be almost identical to the gross rate. On the other hand, if a location has a small population, the gross rate estimation will have a large variance and a small weight will be assigned to this unstable rate; therefore the bayes rate will become closer to the value expected for a randomly selected area in the region.
It is also possible to adjust the rates based on the neighborhood information only.

To estimate, use the Analysis - Local Empirical Bayes option. The Local Empirical Bayes window will be displayed to define the input parameters:
  1. In Theme field, select UP theme.
  2. In Population in Risk field, select the table and the column with population data: UP and POP_2000.
  3. In Number of Cases field, select the table and the column with data of numerator rate (in this cas, homicides in 2000): hom_2000_Collect and Crimes2000.
  4. In Multiplicative Rate Correction field, select 1.
  5. Click on OK to create the new column, to be called LEBayes. If you wish, change its name by typing in the New Column Name field.



In table UP, a new column will be presented: LEBayes, with the values of Local Bayes Rates for each UP.
