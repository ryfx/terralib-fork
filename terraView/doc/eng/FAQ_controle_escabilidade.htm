TerraView - Frequently Asked Questions

How is the visualization of labels of a theme objects controled?

  1. In the theme popup menu, select Visual | Text option and choose text color, size and font; it is recommended to use the Fixed Size option to ensure that the font size will not change when a zoom is applied;

  2. In the same menu, select Text Representation, and choose the attribute to be used to generate the text and the table where the text will be stored;

The range of visualization scales can be controled.

  1. In the theme popup menu, select Visibility of Representations option, type in the Minimum and the Maximum scales and click on OK to execute the operation.
