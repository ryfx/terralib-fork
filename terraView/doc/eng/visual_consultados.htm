Query Visual

This is the presentation visual of theme objects that were queried through attribute or spatial queries. To change the visual of queried objects, right-click to open the popup menu of the theme and select Visual - Query, as shown below.


Changing the Query Visual of a theme

1. Click on the Color button and the window with the color palette will be displayed to select the new color.
2. After choosing the color, click on OK button to return to the main window and click on OK button of that window.
3. To check the result, redraw the theme.
