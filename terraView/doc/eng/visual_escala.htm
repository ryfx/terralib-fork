Graphic Scale Visual

In TerraView it is possible to change the visual of the graphic scale displayed in the Draw Area. This operation is enabled only if the graphic scale is shown. Right-click on the Draw Area and the context menu will be displayed. Click on Graphic Scale Visual option. The window shown in the following figure will be displayed. The single enabled tab in the visual interface is Texts since the graphic scale has this representation only.

1. Click on Color and change text color.

2. Click on Font and change font type, style, and size of texts.

Fixed Size option indicates that the text size will not change with the visualization scale factor. However, this option is not active for graphic scale visual because the size of a scale text remains unchanged during navigation in the Draw Area.

3. Observe in Resulting Visual box a sample of how the text will be presented.

4. Click on OK.

5. Draw Area will be automatically updated and will display the graphic scale with its new visual.
