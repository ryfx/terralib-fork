Pointing Visual

This is the presentation visual of the theme objects that were pointed by clicking in the Draw Area after the tool has been selected or by clicking on the grid of the active theme.
When the Pointing option is selected, the following window will be displayed.


Changing the Pointing Visual of a theme

1. Click on the Color button, the window with the color palette window will be presented to choose the new color.
2. After selecting, click on the OK to return to the main window and click on the OK button of that window.
3. To check the result, redraw and select the desired polygon.
