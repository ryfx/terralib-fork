Intersection Method

Intersection operation uses a theme with polygons type geometric representation to be the clip mask. The theme to be clipped must have geometries of polygons, lines, points, and cells type. Text type geometries are not clipped by this operation.

Themes with geometric representation points or cells type generate a new layer with the objects intersecting the clip mask. Themes with geometric representation polygons or lines type generate a new object for each intersection region.

In the example presented below, a new theme with Belo Horizonte city neighborhoods that intersect or are contained inside the South Region of Belo Horizonte will be created.

1. Select the theme with objects to be selected by intersection;

2. Select the theme with clipping mask;

3. Define the name of the theme to be created;

4. Execute by clicking OK.
