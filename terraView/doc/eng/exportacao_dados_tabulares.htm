Exporting Table Data

Data about map objects may be exported to be used by another application. TerraView provides various export options: whole table, some columns, selected objects, and pointed objects.

1. Activate a cell popup menu and choose Table > Export Table.

Export Table interface will be shown:

2. Choose which part of the table will be exported, by giving the criteria to be used for lines and columns selection. In this case, the whole table will be exported.

3. In the Database frame must be provided the parameters of the Database that will contain the table with the exported data. Its database type is always the same of the active database, and if it already exists, it shall not be a TerraView database. Select the Create option.

4. Select the Database directory where the new database will be created.

5. Define a name for the database (IBGE).

6. Define a name for the table (SetoresCensitarios).

A new table and a new database will be created and can be used in other applications. To visualize the data, open the created database with the associated database manager.

Start Access application to check that the table was created.

Create a new table with HOM_0003 and REN_MED columns only.
