Database Manipulation

Every work session in TerraView requires that at least one Database to be active. Figure 1 shows the interface that controls database manipulation by TerraView.

Creating a Database


Figure 1 - Database manipulation interface.

1 - To create a new database, select Create option in the interface Operation box.

2 - In Database Type list, select the manager used to store the database. Depending on the selected manager the interface will enable and disable other interface items (see Figure 2):


Figure 2 – Manager selection.

3 - ACCESS manager requires the selection of where the database will be created by clicking Diretory button.

4 - Other managers such as SQL Server, OracleSpatial, OracleAdo, MySQL, PostgreSQL, and PostGIS, given that they are database servers, require other parameters to be selected (as shown in Figure 3).


Figure 3 - Database server parameters.

The fields to be filled in :
  • Host: name or address of the computer where the server is installed.
  • Port:port number to create the connection.
  • User:user name to access the server.
  • Password:user password to the server.
  • Define a name for the new database in Database Name field.
  • Click OK.
After a database is created it is automatically connected.

Opening a connection to a previously created Database
    1 - To access a previously created database select Connect option in the interface Operation box (see Figure 4).


    Figure 4 - Connection to previously created database.

    2 - In Database Type list select the manager used to store the database.

    3 - Depending on the selected manager or database type fill in other interface fields:
    • ACCESS Manager: click on Select Database button and select the .mdb file with the TerraLib database. Note that the database name will be shown in the near field.
    • Other managers:
      • Fill in Host, Port, User, and Password fields in a similar way to a new database creation.
      • Click Select Database button. The near list will be filled with the databases available in this server.
      • Select the database to connect.
    3 - Click OK.
Note: if the Check Consistency in the Database item is selected when a database is connected, TerraView will scan the database and remove inconsistencies and lost tables.

Connected databases are shown in the database are of the main interface as shown in Figure 5.


Figure 5 - Database area of the main interface.
