Themes Add Operation

This operation adds themes to create a new layer. Themes must have the same type of vector representation and one type de geometry: polygons only, lines only, points only, or cells only.

In this operation a theme is selected to act as a guide. The guide theme is used as a base. Name and type of the new attributes table are defined from this theme. In addition, the guide theme defines the projection of the new layer geometries. Figure 1 shows three themes to be added:


Figure 1 - Themes to be used in Add operation.

The result of adding these three themes is shown in Figure 2:


Figure 2 – Result of an Add operation.

The created layer contains a copy of all geometries of all input themes. Geometries in different projections are re-projected to the guide theme projection.

The first field of the new attributes table is the primary key and it links to geometries. The remaining attributes are copies of the guide theme attributes. Attributes from other themes used in the operation are added to the table if the name and the type of the attributes coincide with the ones in the guide theme.

It is also possible to add only pointed or queried objects to obtain a map with only the desired objects with their geometries and attributes.

Themes Add Interface

The themes add operation interface is shown in Figure 3 and to activate it use the context menu on a View and select Geoprocessing - Add. In Representation Type field, select the type of geometry to be added. Choose polygons, lines, points, or cells. After the selection, the interface fills in the Input Themes list with themes having the selected geometric representation.


Figure 3 - Themes Add Operation interface.

In Input Themes list select the themes to be added. Objects to be added may be all of them, pointed ones only, or queried ones only.

In Fields list select the theme of reference. The output layer table will contain all attributes of this theme. Attributes of other themes with the same name and type of a reference theme attribute will be added to the output layer table.

Output Layer Name box is automatically filled in by the interface, but it can be edited to create the layer with another name.

OK button applies the add operation.

Cancel button disables the interface.

Help button activates the help interface.
