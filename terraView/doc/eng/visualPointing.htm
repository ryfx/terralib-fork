TerraView

Pointing Visual

TerraView allows pointing on the map and finding out attributes of the pointed object in the grid, or pointing on the grid and finding out the location of the pointed object in the map.

Pointing visual color is defined in such a way that pointed objects are visually differentiated from other objects.

Note that only the color is defined. Other drawing attributes, such as polygon filling style, line style, size and type of points are maintained. The visual highlight is applied only by color.

Visual color is also used in the grid to display the objects pointed in the map. The corresponding lines are drawn with this color.

Pointing visual color is defined by the color selection interface.
