Own Visual

Visual of an individual object of a theme can be changed. In the Draw Area, right-click on the object you wish to change its visual. A context menu will be shown. Click on Own Visual - Set the Own Visual of the Object... . The window presented in the following figure will be shown. Tabs of the window are enabled accordingly to the representations contained in the theme.

1. Change the visual using the procedure explained in the Default Visual item.

2. Click on OK.

3. Draw Area will be updated automatically and will display the object with its new visual.

Repeat this procedure for all objects you wish to change their visual. To display the objects with the default visual, either click on Own Visual - Remove Own Visual to remove the visual of a given object or click on Remove Own Visual of All Objects in the context menu of the Draw Area.
