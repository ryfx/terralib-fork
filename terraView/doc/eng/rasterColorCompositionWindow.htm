Raster Color Composition

Creating various compositions with bands allows the identification of features that may not be visible in the original composition.

Select the raster theme you wish to use, right-click to display the pop-up menu, and select the Properties of Image Display - Color Composition option.

To create a monochromatic composition, select the corresponding band in Mono field and select OK.





To create other compositions, select the band that corresponds to each color using the lists on the right and select OK.
