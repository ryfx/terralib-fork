Creating Views and Themes


Activate *.mdb Database.

To create a View called BH_Novo, with all imported layers to be visualized at once, execute the following steps. Use the shortcut from the Tool Bar and name the view as BH_Novo. To visualize any map from a database, a theme must be added to the view. Bairros layer will be added to the active BH_Novo View. In order to achieve that use the shortcut or follow the command Theme > Add.


A new window will be displayed and used to:

1. Define the layer to be used. Select Bairros.

2. Define to which view the theme will be added. Select BH_Novo.

3. Select Theme Table(s).

4. Name this Theme. Type in Bairros.

5. Execute the procedure.

Observe that the Bairros theme is added to the View called BH_Novo. With this Theme activated, a map can be visualized in the Draw area and table information related to it will be available in the Grid area. See Figure below.

Now add the themes related to Colegios and Favelas layers to the BH_Novo View.

Using and buttons, previous and next visualizations in relation to the current one are drawn. If no change was applied to the visualization, the buttons are disabled.

Reset button is useful and redraws the map by sizing it based on the Draw area size to display all visible themes.
