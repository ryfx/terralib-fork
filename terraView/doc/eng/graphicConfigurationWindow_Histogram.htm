Histogram Graphic Configuration

The interface for configuration of a Histogram Graphic is shown in Figure 1. The histogram is drawn using the colors defined for the theme. Pointed (selected) objects by the mouse are drawn with pointing color of the theme and the not pointed objects are drawn with the theme default color.


Fig. 1 - Interface for configuration of a Histogram Graphic.

The interface contains a field to change the color and style of the histogram outline.

Color... button activates the color selection interface.

Style list selects line style among:
  • NoPen,
  • SolidLine,
  • DashLine,
  • DotLine,
  • DashDotLine, and
  • DashDotDotLine
