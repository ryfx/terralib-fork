Create Layer Table
TerraView allows the creation of a table in a layer by the user.
Activate the context menu on MG_cities layer and select Create Table option. The Create Table interface will be displayed.




Define a name for the new table (New_Table). The new table has been associated to MG_cities layer and can be used to store new information.

To associate the table to the theme, activate the context menu on MG_cities theme and choose Select Theme Tables option. Insert the new New_Table table in MG_cities theme. Observe that only the new column has been inserted.
