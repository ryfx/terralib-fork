Address Name Separator

This window is only shown if the field associated to the address number was not provided, when of configuring the table with addresses to be geocoded. In this case, it is assumed that both the address number and address name are in the field associated to the address name, separated by some character.

You must type in the appropriate text box the character to be used as a separator between the address name and the address number. After that, click on OK button.

You can just return to the previous interface by clicking on the Cancel button.
