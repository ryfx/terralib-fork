Open Database

To open (or to connect to) a database, select the database option from the shortcut or from the menu File > Database. A Database window will be opened and there are the options of creating a new database or connecting to an already existing database. The example below shows the required steps to open (or to connect to) an already existing database.

1. Choose Connect option.

2. Define the type of database to be connected. TerraView uses various DBMS such as, Access, Oracle and MySQL. Select Access option.

3. Select Bancos directory where the database was saved.

4. Choose the database to be connected.

5. Execute the previous procedures.
