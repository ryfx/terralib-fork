Bar and Pie Charts

Bar and pie charts allow users to evaluate comparatively two or more attributes of an individual object. To display the Edit Bar or Pie Chart window, right-click on the districts Theme and select Edit Bar or Pie Chart option.

To create a chart, execute the following steps:

  1. Select the graphic Type: bar chart or pie chart. In this example select Bar Chart.
  2. Select the attributes to be used in the chart in Chart Items box. Select the attribute from the Attribute list and click on Insert button. To remove a selected attribute, click on the row number and click on Remove button. In this example, select the POP_FAVELADA and MORT_INFANTIL attributes.
  3. Define the chart size:
    • For a bar chart, define Minimum and Maximum Height, and Width (in number of pixels);
    • For a pie chart, define Minimum and Maximum Diameter (in number of pixels);
    • Fixed Size is used if the chart items size (bar and pies) is not going to change with zoom operations.


  4. For a pie chart, an attribute can be selected to define the diameter to be Proportional to.
  5. Input field allows the creation of bars/pies to only some objects of the Theme.
  6. Click on OK

Using the charts, it is possible to analize two variables, observing the proportion between them individually by district. Right-clicking on the name of the bar and pie charts or on each item, will access the options to re-build, remove, and define other names for the chart variables. The procedures presented below to display and edit chart legends in the Draw Area are also valid.
