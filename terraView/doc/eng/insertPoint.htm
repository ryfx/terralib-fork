Insert Point

Terraview allows points to be inserted directly in the Draw Area. This operation is enabled only for themes with point representation. Right-click on the Draw Area at the position where the point is to be inserted and the context menu will be displayed. Click on Insert Point option. The interface shown in the following figure will be displayed.

1. An Object Identifier is suggested and, if you wish, can be changed to a more convenient one.

2. Select the type of coordinates: Plane or Geographical. Change the coordinates Long and Lat if you wish.

3. Click on OK.

4. Draw Area and Grid will be automatically updated to display the inserted point and the record with the point and the selected identifier.
