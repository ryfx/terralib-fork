Spatial Query

Besides pointing selection, it is possible to query the database directly by establishing spatial relationships between the objects of a same theme or two different themes. TerraView offers a tool for spatial query, for example, to visualize all the crimes committed in a particular neighborhood or all the neighborhoods that border a certain neighborhood.

Spatial functions

TerraView can use spatial functions to work with one particular theme.

1. Activate Bairros theme.

2. Select the objects to apply the function before executing query. Select Centro and Sion.

3. Activate popup menu from Bairros theme and select Spatial Query.

4. Select the function to be executed in Spatial Functions field. The available functions depend on the database type and the geometry in the theme. For Access database the following options are available:

Function Points Lines Polygons
Distance Distance between two selected points Distance between two selected lines Distance between two selected polygons
Area No No Area of selected
Length No Length of selected Perimeter of selected
Intersection No No Intersection of selected
Difference No No Difference between two selected polygons
Union No No Union of selected
