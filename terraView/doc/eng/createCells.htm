Create Cells Layer

A layer of cells is created from another layer. TerraView has two strategies to create cells depending on the type of spatial data of the input layer. If the type is raster or another geometry that is not polygonal, the cells will be created from the bounding box of the layer. In this case, cells will cover the whole extents of the layer, in a way similar to a raster data, with a constant number of cells in each row and column of the layer. If the type of the data is polygonal, the cells will be created in a way that covers the polygons, but cells will be created only where there is an intersection between a polygon and a created cell.

When menu item Layer - Create Cells is clicked or the context menu on the layer name is clicked and the Create Cells option is selected, the interface shown in the figure below will be displayed.

1. Select the reference Layer or Theme to create the cells layer. Observe that the interface ticks the Polygons option if it finds out that the selected layer has this representation.

2. The fields related to Bounding Box are filled in based on the dimension of the selected layer or theme.

3. In New Layer field, fill in Res. X and Res. Y based on the reference layer scale and on the number and size of the cells to be created. Type in a Name for the new cells layer.

4. Click on OK. A warning message will be displayed to inform the number of columns and of rows of the cells layer to be created. Click on Yes to confirm. This operation may take a long time depending on the selected layer size and the number of cells to be created.

When the operation is finished, the new layer with cells has been created. Create a theme using this layer to visualize it in the Draw Area.
