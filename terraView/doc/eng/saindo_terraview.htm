Exiting TerraView

In TerraView, all operations save information automatically in the database. This implies that it is not necessary to save any file when exiting from the program. Everything is already saved in the database: Layers, Views, Themes, and External Tables. On the other hand, if a created Layer, View, or Theme is not required to be saved, the user should delete them explicitly.

The information is already saved. Exit TerraView.
