TerraView 3.3.1

Intersection Operation Between Themes

Intersection operation uses a theme with geometric representations of polygons type as an overlay mask. The theme to be clipped may have geometries of polygons type only, lines type only, points type only, cells type only, or raster type (images or grids).

For themes with geometric representations of point type or cell type, a new layer is created with objects that intersect the overlay mask.

For themes with geometric representations of polygon type or line type, a new object is created for each intersection region. An object may be divided in many parts and a new object is created for each part. Divided objects have the same attributes of the source objects, although they have different identifiers. If required, a grouping operation can be executed to create an object similar to the source one.

For themes with geometric representations of raster type, a new layer is created with valid values only where there are intersections with the overlay mask. All source raster representation characteristics are preserved in the clipped representation.

The attributes table of the created layer may have the attributes of the overlay mask theme in addition to the attributes of the source theme.

The projection of the created layer is the same of the clipped theme layer. When the overlay mask theme has a different projection, the geometries are reprojected in order to operate in the projection of the clip theme.

Intersection Between Themes Interface

Intersection between themes interface is shown in Figures 5 and 6. To activate the interface use the context menu on a View and select Geoprocessing - Intersection. In Select Input Theme to be Clipped by Intersection, select the theme to clip. If the theme to be clipped has vector representations, the interface changes to the one shown in Figure 5.


Figure 5 - Intersection between themes interface for vector input.

Use input theme selection box buttons to set the objects to be clipped: all, pointed only, or queried only.

When a theme to be clipped has raster representation, the buttons to select objects are disabled.

 

Select a Overlay Polygon´s Theme box is used to choose the theme to be used as overlay mask. Use the buttons to set the objects to be used as mask: all, pointed only, or queried only.

In addition, choose if the output layer table will include overlay theme attributes by selecting the Put Overlay Theme Attributes option. This option is available only for themes to be clipped with vector representation, otherwise it is disabled.

Specify the Output Layer Name box is automatically filled in by the interface, but it can be edited to create the layer with another name.

To clip themes with raster representation, set Background Values field with the value to be used to fill areas not intersected by polygons of the overlay mask. Figure 7 shows the difference between using 255 and 0 as background value.


Figure 7 – Images clipped with different background values.

OK button applies the intersection operation.

Cancel button disables the interface.

Help button activates the help interface.
