Associating Files and Internet Sites to Objects on the Map

It is possible to associate files or Internet sites to objects on the map. For example, the easy access to Sion Neighborhood Residents Association may be accessing Sion neighborhood.

1. Select the neighborhood (by line or by object).

2. Activate the Line popup menu, and choose Insert URL.

3. Write the homepage address e.g.: (www.amesion.com.br) and then click OK.

Now, in order to find the Sion Neighborhood Residents Association homepage follow the same steps as in Displaying Information about Objects.

1. Select Information Cursor.

2. Click on the map the Sion neighborhood area.

3. Information about the selected area.

4. Double click on Sion Neighborhood Residents Association homepage address (www.amesion.com.br).

Likewise, it is possible to insert any file related to the object, for example, a photo that reminds you of the neighborhood. To do so, choose Insert Media option.

It is also possible to choose the Default Media Description option. This is a space reserved for the description of the resource inserted into the map.

Files and sites related to the object can also be removed:

1. Right click on the line of a neighborhood and choose Attributes and Media option.

2. Click on the field where the media is.

3. To remove media or URL, choose Remove option.

4. In case there is more than one media associated to a table line, with the Default Media Description option, the line to be shown will be selected with a double click on it.
