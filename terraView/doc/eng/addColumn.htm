Add and Remove Column Data

 

Table columns can be added or removed.

1. Activate the context menu on the header of the POP2000 column and choose Add Column option.

2. Define the table where the new column will be created (Nova_Tabela).

3. Define type of data as Real for this column. If String option is selected, then choose also the size of the column.

4. Type in the name of the new column (den_pop).

5. Click on OK button.

Inserting data in a column

1. Activate the context menu on the header den_pop column, which has just been created, and choose option Change Column data option.


To calculate the populational density in year 2000 for each neighborhood, use POP2000 column.

2. Define the operation used to create the new data for the column by selecting POP2000 column, ratio ( / ) operator, and AREA column.

3. Click on OK.

Note that Xden_pop column has been filled in with the results from the operation of step 2.

Removing columns and tables

1. To remove a column, activate the context menu on the header of the column to remove and choose Remove Column option.

2. To stop the visualization of a table for a theme, activate it, select its context menu, and choose Select Theme Tables option.

3. To remove tables from a database that were created or associated to a layer , activate the context menu of Bairros layer and choose Remove Layer Table (Nova_Tabela).
