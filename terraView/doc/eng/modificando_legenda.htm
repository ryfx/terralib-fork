Modifying Legend

A legend group intervals, colors, and names, in addition to the legend name can be modified. To modify, use Legend Editor, available from Edit Legend option in theme popup menu or Bairros.POP2000 legend popup menu.

To exclude a legend, activate legend popup menu and select Remove Legend option.

Legend Name Modification

A legend name can be modified.

1. Activate popup menu of legend (Bairros.POP2000 ) displayed with Bairros theme.

2. Select Set Alias option.

3. Define a new name for Bairros2000 legend and press Enter key.

Legend Intervals Modification

Intervals, different from the automatically calculated ones by TerraView, can be set by using values that may be more appropriate. Use Legend Editor:

1. Double click on the interval to change.

2. Type in the minimum and the maximum value of the interval, separated by ~. Be careful since no consistency check will be applied. Interval name will be updated from these values.

Legend Label Modification

The default label is the values interval of each group or, if Count was selected, the number of objects in each group. To modify labels, use Legend Editor by activating it from Edit Legend option.

1. Double click on the label to modify it.

2. Type in the new group name and press Enter key.

Label and color of a group can be modified by activating its popup menu:

1. Right click on the legend group to modify it.

2. Select Rename Legend option. Type in the new group name and press Enter key.

3. Select Change Visual option to modify the group color.

Creation of many thematic maps for the same layer

A theme may have only one legend. A new legend will overwrite the previously existing one. To visualize different thematic maps for the same layer, a new theme must be created and a new legend defined for it.

For instance, to create a legend for neighborhoods by area, create a new theme for the Bairros layer and for these new theme, define a legend.
