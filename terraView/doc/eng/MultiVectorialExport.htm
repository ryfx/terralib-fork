Multiple Vector Export

TerraView allows vector data from several layers or themes to be exported in a single operation. Geographic information with their attributes are exported. The available format to export are: SPRING, MID/MIF, Shapefile, and TXT. This tool is accessed through the File - Multiple Vector Export menu option, which enables export of data from Layers, and through the context menu on a View, which enables export of data from the Themes selected in the Views/Themes Tree.

1. Select a Layer or Theme from the left side list and click on to add it to the list of layers or themes to export. Add as many layers or themes as you wish.

2. Select a Layer or Theme from the right side list and click on to remove it from the list of layers or themes to export. Remove as many layers or themes as you wish.

3. Choose the Format of the file to be created. For TXT format, only layers or themes with representations of point type will be exported.

4. Click on Path and define the destination folder for the files to be created. File names are defined by the type of source (Layer or Theme) and the layer or theme name, for instance, LAYER_Colegios.

5. Check the Export properties option if you want to export the layer properties, such as projection parameters and bounding box. In Themes export, this option is disabled.

6. Click on OK.
