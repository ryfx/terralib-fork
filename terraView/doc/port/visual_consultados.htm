Visual Consultados

Este é o visual de apresentação dos objetos do tema que foram consultados via consulta por atributo ou consulta espacial. Para a mudança do visual consultados, clique com o botão direito do mouse abrindo o menu Pop Up do Tema selecione Visual - Consultados como mostra abaixo.


Mudando o Visual de Apontamento de um tema

1. Clique no botão Cor, como resultado será mostrada a janela da direita com a paleta de cores para escolha da nova cor.
2. Após a escolha, clique no botão OK para voltar à janela da esquerda, e em seguida no botão OK daquela janela.
3. Para verificar o resultado, desenhe novamente o tema.


Mudança no Visual Consultados: antes (esq.) depois (dir.)
