Plugin WMS Cliente para TerraView
Guia do Usuário

Autor: Gilberto Ribeiro de Queiroz (gribeiro@dpi.inpe.br)
Última atualização: 17/09/2010

O plugin WMS Cliente do TerraView é uma ferramenta que possibilita a visualização de camadas de informação a partir de servidores WMS (Web Map Service). Este guia apresenta todas as funcionalidades incorporadas por este plugin.
