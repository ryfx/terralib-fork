Operação de Atribuição de Dados por Localização

A operação de atribuição de dados por localização é executada entre dois temas selecionados. Ela consiste em adicionar os atributos de um tema X a um outro tema Y, baseado na relação espacial entre os objetos geográficos desses temas. Essa operação cria uma nova tabela de atributos contendo os atributos de X e a associa ao tema Y.

Há duas estratégias para essa operação: Distribuição e Coleta. Considerando dois temas, tema X que fornece os atributos e o tema Y que recebe os atributos, as duas estratégias podem ser explicadas abaixo:

A estratégia de Distribuição consiste em distribuir os atributos de cada objeto geográfico do tema X a todos os objetos do tema Y que possuem uma determinada relação espacial.

A estratégia de Coleta consiste em, para cada objeto geográfico do tema Y, coletar ou agrupar os atributos de todos os objetos geográficos do tema X que possuem uma determinada relação espacial.

Por exemplo, considerando dois temas MUNICIPIOS e ESTADOS, onde um representa os municípios (representação geométrica polígono) e o outro os estados do Brasil (representação geométrica polígono), ilustrado na Figura 1. Cada município tem os seguintes atributos: NOME, POPULACAO e ESTADO. E cada estado tem os atributos: NOME e AREA. A seguir será mostrado o resultado de cada estratégia de atribuição de dados utilizando esses dois temas.


Figura 1 - Temas ESTADOS e MUNICIPIOS

Distribuição

 Um exemplo de distribuição de dados por localização sobre os temas ESTADOS e MUNICIPIOS é: distribuir os atributos de cada estado aos municípios que estão dentro dele. Nesse exemplo a relação espacial usada é "está contido" ou "dentro". Essa operação gera uma tabela de atributos associada ao tema MUNICIPIOS, mostrada na Figura 2.


Figura 2 - Tabela de atributos resultante - Distribuição

Na estratégia de distribuição, o tema que recebe os atributos pode ter representação geométrica: Ponto, Linha, Polígono ou Célula. E o tema que fornece os atributos pode ter representação geométrica: Polígono, Célula ou Linha. Coleta

Um exemplo de coletas de dados por localização sobre os temas ESTADOS e MUNICIPIOS é: coletar ou agrupar os atributos de todos os municípios que estão dentro de cada estado. Nesse exemplo a relação espacial usada é "está contido" ou "dentro" e os atributos foram agrupados usando funções de sumarização. Para o atributo NOME foi utilizado o menor valor em ordem alfabética (função de sumarização Mínimo) e para o atributo POPULACAO foi utilizado a soma de todas as populações dos municípios de cada estado (função de sumarização Soma). Essa operação gera uma tabela de atributos associada ao tema ESTADOS, mostrada na Figura 3.


Figura 3 - Tabela de atributos resultante - Coleta

Interface de Distribuição de Dados entre Temas

A interface de distribuição de dados entre temas é mostrada na Figura 4, para acessá-la utilize o menu de contexto sobre a Vista, na opção Operações Geográficas - Atribuir Dado por Localização - Distribuir. Nessa interface o usuário escolhe dois temas, o que irá fornecer e o que irá receber os atributos, através das caixas Selecionar Tema para Atribuir Dados para e Selecionar Tema para Atribuir Dados de.

Além disso, o usuário escolhe a relação espacial que será considerada na operação. As relações espaciais são habilitadas de acordo com a representação geométrica dos temas escolhidos anteriormente.


Figura 4 - Interface de distribuição de dados por localização

A caixa Criar e Atribuir Dados para Tabela contém o nome da tabela de atributos que será gerada e é automaticamente preenchida pela interface. Portanto, pode-se editá-la a fim de criar uma tabela com um nome desejado.

A caixa Atributos a serem distribuídos possui uma lista que permite marcar ou desmarcar, individualmente, qualquer atributo desejado do tema para realizar a distribuição. Ela também possui dois botões que permitem selecionar rapidamente todos os atributos do tema ou nenhum atributo.

O botão Executar efetua a operação de adição.

O botão Cancelar inibe a interface.

O botão Ajuda ativa a interface de ajuda.

 

Interface de Coleta de Dados entre Temas

A interface de coleta de dados entre temas é mostrada na Figura 5, para acessá-la utilize o menu de contexto sobre a Vista, na opção Operações Geográficas - Atribuir Dado por Localização - Coletar. Nessa interface o usuário escolhe dois temas, o que irá fornecer e o que irá receber os atributos, através das caixas Selecionar Tema para Atribuir Dados para e Selecionar Tema para Atribuir Dados de.


Figura 5 - Interface de coleta de dados por localização

A caixa Criar e Atribuir Dados para Tabela contém o nome da tabela de atributos que será gerada e é automaticamente preenchida pela interface. Portanto, pode-se editá-la a fim de criar uma tabela com um nome desejado.

Além disso, o usuário escolhe a relação espacial que será considerada na operação. As relações espaciais são habilitadas de acordo com a representação geométrica dos temas escolhidos anteriormente.

A caixa Atributos de Saída possui uma lista que permite marcar ou desmarcar, individualmente, qualquer estatística desejada de qualquer atributo do tema. Ela também possui dois combos que permitem selecionar rapidamente, por exemplo, todos os máximos, ou mínimos, ou somas etc. de todos os atributos do tema.

O botão Executar efetua a operação de adição.

O botão Cancelar inibe a interface.

O botão Ajuda ativa a interface de ajuda
