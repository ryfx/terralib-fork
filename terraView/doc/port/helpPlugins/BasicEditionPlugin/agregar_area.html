Agregar Área
Esta ferramenta é usada para aumentar a área de um polígono, acrescentando a área de outro polígono a ser desenhado, que se sobreponha ou pelo menos toque o polígono original. O resultado será um novo polígono, que é a união dos dois polígonos, com os atributos descritivos do polígonos original.

Agregar Área :
  1. Selecione um polígono.
  2. Selecione a ferramenta agregar área.
  3. Desenhe a área a ser agregada, passando pelo polígono selecionado.
  4. Para finalizar clique com o botão direito do mouse.
  5. Para Salvar, selecione o ícone
