Juntar Linhas
Esta funcionalidade opera sobre duas linhas, fazendo com que passem a ter uma extremidade em comum, mas preservando a identidade de cada uma, que continua como linha independente da outra. Inicialmente deve-se selecionar ambas as linhas.

Juntar Linhas: Habilita o cursor para juntar as linhas que estejam selecionadas, fazendo com que estas possuam uma extremidade em comum.
  1. Selecione a primeira linha.
  2. Pressione o botão CRTL do teclado e selecione a segunda linha.
  3. Clique ao redor das extremidades das linhas onde será feita a junção.
  4. Arraste o mouse entre as linhas selecionadas de modo que as extremidades de ambas fiquem dentro do quadrado.
  5. Para finalizar, selecione o botão Salvar
