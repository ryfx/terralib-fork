Criar Polígono
Para que essa ferramenta seja habilitada, o tema ativo deverá ser associado a um plano de informação com representação vetorial de polígonos.
  1. Para criar um polígono, clique no ícone:
    • Criar Polígono Ponto a Ponto: : habilita o cursor para criar um polígono ponto a ponto. Para finalizar o polígono, o usuário deverá clicar com o botão direito do mouse, fazendo com que o último ponto seja automaticamente ligado ao ponto inicial.
    • Criar Polígono Contínuo: : habilita o cursor para criar um polígono de forma contínua. Para finalizar o polígono, o usuário deverá clicar com o botão direito do mouse, fazendo com que o último ponto seja automaticamente ligado ao ponto inicial.
  2. Clique no ícone Criar Polígono Contínuo.
  3. Clique com o botão direito do mouse para o fechamento da geometria. A cada geometria desenhada, será apresentada a janela de atributos para que possam ser informados os atributos descritivos associados àquela geometria.
  4. Defina um valor para a coluna de ligação no campo Coluna de Ligação.
    • Quando fornecido um valor já existente para a coluna de ligação, ao clicar no botão Validar serão carregados na janela de atributos os valores armazenados correspondentes àquele valor; Caso o usuário Salve os valores carregados, a nova geometria será também associada àquele registro, ou seja, haverá mais de uma geometria associada com um dado registro alfanumérico.
    • Caso deseje utilizar o Valor da Coluna de Ligação Anterior selecione o botão Usar Atributo Anterior. Este caso é similar ao anterior, pois haverá também mais de uma geometria para um mesmo registro e os valores existentes serão carregados na janela, sendo útil quando geometrias de cada grupo são desenhadas em sequência, facilitando a associação com um mesmo elemento.
    • Caso deseje que o sistema gere automaticamente um valor para a coluna de ligação selecione o botão Gerar Identificação Automaticamente. Este é o caso mais comum, quando cada geometria possui um conjunto de atributos específico, e não é necessário se preocupar com os valores para a coluna de ligação.
  5. Ao preencher a janela de atributos ou ter digitado um valor para a Coluna de Ligação, clique no botão Salvar, serão salvos os atributos descritivos informados. A seguir o polígono será apresentado na área de visualização.
