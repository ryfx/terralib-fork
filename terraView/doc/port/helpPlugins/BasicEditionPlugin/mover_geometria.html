Mover Geometrias
Para a utilização dessa ferramenta, selecionar a ferramenta

Mover Geometrias :
  1. Selecione a(s) geometria(s), para selecionar mais de uma pressione o botão Ctrl do teclado.
  2. Selecione a ferramenta Mover Geometrias.
  3. Arraste a geometria até o novo local. No caso de várias geometrias, basta arrastar uma delas que as demais sofrerão o mesmo deslocamento.
  4. Para Salvar, selecione o ícone
