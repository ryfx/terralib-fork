TerraView - Perguntas mais frequentes

Como reposicionar, remover ou alterar o tamanho de um rótulo de objeto?

Rótulos de objetos são textos usados para identificação visual dos objetos. Eles podem ser gerados usando uma ferramenta disponível no TerraView. Para isso, veja a pergunta Como gerar texto associado a atributos de um plano de informação?

Para reposicionar um rótulo, selecione o ícone E (Edição do Tema) na barra de ferramentas, e em seguida o cursor de apontamento (seta). O próximo passo é selecionar o rótulo, para isso posicione o cursor seta sobre ele, clique e mantenha pressionado o botão esquerdo do mouse, verá que o cursor muda de seta para uma cruz; ainda com ele pressionado, arraste o mouse sobre o rótulo, formando um quadrado ou um retângulo, e libere o botão. Ao ver que o rótulo ficou selecionado, coloque o cursor sobre o centro do mesmo até aparecer uma mãozinha, e o arraste clicando no botão esquerdo do mouse e movimentando para o local desejado, liberando o botão quando concluir.

Para remover, selecione o rótulo usando o procedimento anterior de seleção, e aperte a tecla Delete do seu teclado.

Para alterar o tamanho do rótulo, repita o procedimento anterior de seleção e coloque o cursor sobre a borda do rótulo, verá o cursor mudar para uma seta. Clique no botão esquerdo do mouse e arraste lentamente, aumentando ou diminuindo o rótulo, e libere o botão para concluir a operação.

Observação 1 : A operação de seleção de rótulo pode ser feita para selecionar diversos rótulos ao mesmo tempo, definindo-se uma área que engloba todos os rótulos desejados. Isso é feito clicando no botão esquerdo do mouse e arrastando o mouse enquanto o cursor se encontra no formato de cruz. Com isto, é possível reposicionar, remover ou alterar o tamanho de vários rótulos ao mesmo tempo.


Observação 2 : A operação de alteração de tamanho de um rótulo de objeto não fica disponível para os rótulos gerados com a opção de visual de texto "Tamanho Fixo" selecionada. Esta restrição perdura mesmo que, após a geração dos rótulos, a citada opção seja desselecionada.
