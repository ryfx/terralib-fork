Consulta Simples

 

Abra o banco de dados BH.mdb. Você irá verificar que este banco contém as camadas de AreasVerdes, Bairros, Batalhões, Colégios, CompanhiasPM, Crimes2000, Crimes2001, Crimes2002, Crimes2003, Favelas, Lagoa, Limite, PostosCombustivel, Rios, RuasCentrais, SetoresCensitarios e UP. Ative o tema Bairros e verifique na janela de grade os atributos associados a este objeto geográfico. A coluna de nome “POP2000” representa a população no ano de 2000, correspondente a cada bairro. Você poderá, usando uma consulta por atributo, selecionar os bairros com população maior que 6000 habitantes.

1. Ative o tema Bairros e deixe apenas ele visível;

2. Use o menu popup;

3. Escolha a opção Consulta por Atributos...

A janela Consulta por Atributos é aberta.

1. Defina o atributo que será usado para a seleção. Escolha POP2000.

2. O campo Operador define qual tipo de operação você deseja realizar sobre o atributo. Selecione >. Os operadores disponíveis no TerraView são:


3. No campo Valores do atributo você define o valor a ser utilizado para comparação. Se você selecionar essa opção, todos os valores encontrados para essa coluna irão aparecer para escolha no campo à direita. Você também pode optar por entrar com o valor diretamente no campo de edição. Selecione o botão e escolha o valor 6000.

4. Clique no botão Nova Consulta.

Você irá perceber que algumas linhas foram destacadas na janela de grade. Essas linhas são aquelas correspondentes ao resultado da consulta: os bairros que tem população maior que 6000 habitantes foram selecionados.   • Alguns operadores comparativos

    Além dos operadores comparativos usuais, como =, >, <, <>, <=, >=, o TerraView disponibiliza três outros operadores muito úteis: LIKE, IS NULL e IS NOT NULL.
    A seguir você verá a função desses operadores e alguns exemplos de sua utilização.

    LIKE

    O operador comparativo LIKE pode ser utilizado quando se deseja consultar todos os valores de uma determinada coluna que possuam determinados caracteres em comum.

    Use o menu popup do tema Crimes2003 e selecione a opção Consulta por Atributos...
    Defina o atributo como NAT, o operador como LIKE e clique no campo de edição. Digite 'C%' na seqüência do operador. Clique no botão Nova Consulta.

A consulta mostrará todos os valores da coluna NAT que se iniciam com C. No caso, a coluna NAT apresenta os códigos dos crimes ocorridos em Belo Horizonte no ano de 2003 e aqueles que se iniciam com a letra C referem-se a um Roubo. Verifique na janela de grade que todos os tipos de roubos foram destacados. A entrada do operador (no caso 'C%') reflete como deve ser feita a consulta. Essa entrada deve estar entre apóstrofe (' ') ou aspas (" "); o sinal de porcentagem (%) indica que qualquer conjunto de caracteres (incluindo a ausência deles) pode estar ocupando aquela posição. Assim, no exemplo utilizado, consultam-se todos os valores que se iniciam com o caractere C. Caso fosse de interesse consultar os valores que terminam com o caractere 7, o comando seria NAT LIKE '%7'; se desejar consultar os valores que possuam o caractere 2 em qualquer posição, o comando seria NAT LIKE '%2%'; se, por fim, não quiser que constem nessa última consulta aqueles valores que terminem em 2, digita-se o comando NAT NOT LIKE '%2' e clica-se no botão Filtra.


IS NULL

O operador comparativo IS NULL pode ser utilizado para consultar os valores de uma determinada coluna que possuem valores ausentes. Use o menu popup do tema Colegios e selecione a opção Consulta por Atributos... Defina o atributo como ENSINOMEDI e o operador como IS NULL. Esse operador não necessita de entrada, não necessitando adicionar mais nada ao comando. Clique no botão Nova Consulta.

Os colégios que oferecem o Ensino Médio estão marcados com um 'X'e os demais não possuem valores na coluna ENSINOMEDI; a consulta destaca, portanto, aqueles colégios que não oferecem Ensino Médio.

IS NOT NULL

O operador comparativo IS NOT NULL desempenha a função inversa do operador IS NULL: consulta as linhas para as quais existem valores naquela coluna. Use o menu popup do tema Favelas e selecione a opção Consulta por Atributos... Defina o atributo como POPULAÇAO e operador como IS NOT NULL. Também não será necessário digitar uma entrada. Clique no botão Nova Consulta.



A consulta mostrará aquelas Vilas/ Favelas, Conjuntos Habitacionais, Bairros e Aglomerados para os quais há valor de população. Essa mesma consulta pode ser feita como NOT POPULAÇAO IS NULL. Acrescentando-se NOT no início de um comando, faz-se a consulta inversa.
