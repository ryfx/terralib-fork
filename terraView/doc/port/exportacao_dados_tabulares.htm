Exportação de Dados Tabulares

 

Você pode querer exportar dados dos objetos do mapa para trabalhar em outro aplicativo. O TerraView fornece várias opções de exportação: a tabela inteira, algumas colunas, objetos selecionados ou apontados.

1. Ative o menu popup de célula e escolha a opção Tabela > Exportar Tabela.

A janela Exportar Tabela será aberta:

2. No quadro Seleção são indicadas quais linhas e colunas da tabela se deseja exportar. Você irá exportar a tabela inteira.

3. No quadro Banco de Dados são informados os parâmetros do Banco de Dados. O tipo de Banco de Dados é o mesmo do Banco de Dados que está ativo. Você pode se conectar a um banco de dados já existente, mas não a um banco TerraView. Escolha a opção Criar.

4. Escolha o diretório Bancos onde seu novo banco de dados será criado.

5. Defina um nome para o seu banco de dados (IBGE).

6. Defina o nome da sua tabela (SetoresCensitários).

A nova tabela e o novo banco serão criados e você poderá utilizá-los em outros aplicativos. Para visualizar as tabelas, você deve abrir o banco criado como o gerenciador associado.

Ative o aplicativo Access e verifique que sua tabela foi criada.

Crie agora uma nova tabela somente com as colunas HOM_0003 e REN_MED.
