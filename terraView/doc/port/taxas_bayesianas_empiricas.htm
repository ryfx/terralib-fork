Taxas Bayesianas Empíricas

 

Nesta seção, você irá construir taxas (razões) a partir de atributos dos objetos dos mapas. Primeiramente, você irá calcular a taxa bruta de homicídios ocorridos no ano 2000, para bairros de Belo Horizonte. Em seguida, usaremos a ferramenta de Bayes Empírico Global, para ajustar as taxas.

A taxa bruta é uma divisão entre um numerador (número de homicídios, seqüestros, mortes) e um denominador (população em risco, em geral). A taxa bruta apresenta grande instabilidade para regiões com população de risco pequena, isto é, regiões com pequenas populações estão muito mais suscetíveis às flutuações nas taxas devido à ocorrência de uns poucos eventos por mero acaso do que regiões populosas. A taxa bayesiana empírica global baseia-se no uso de informação das outras áreas que compõem a região de estudo para diminuir esta instabilidade não associada ao risco de ocorrência do evento.

O primeiro requisito do TerraView para calcular as taxas Bayesianas é que o denominador (a população de risco) não contenha valores menores ou iguais a zero. Caso não houvesse dados sobre população para alguns bairros de Belo Horizonte, seria necessário, portanto, criar uma camada que contivesse apenas bairros para os quais temos esses dados.

Calculando a taxa bruta

Para calcular a taxa bruta, crie uma nova coluna na sua tabela de bairros (Tema Bairros) e atualize os dados desta coluna. Para criar uma nova coluna, use o menu popup a partir do cabeçalho da coluna na sua janela de grade e escolha a opção Adicionar Coluna.

Depois de criar a coluna chamada de Taxa, use o menu popup a partir desta coluna e altere seus dados:

Você deve fazer uma operação onde o numerador é o número de homicídios no ano de 2000 e o denominador é a população dos bairros de BH no mesmo ano. Multiplica-se por 100.000, apenas para padronizar os valores.

Calculando a taxa Bayesiana empírica

Você irá, agora, criar uma coluna com a taxa bayesiana empírica global, para ajustar as taxas para municípios de população pequena. Para isto, use o menu Análise e depois Bayes Empírico Global... A janela para entrada de parâmetros será aberta:
Informe os seguintes parâmetros:

1. No campo Tema, selecione o tema desejado Bairros.

2. No campo População em Risco, selecione a tabela e a coluna que contém os dados de população: Bairros e POP2000.

3. No campo Número de Casos, selecione a tabela e a coluna que contém os dados do numerador da taxa (no nosso caso, homicídios em 2000): Bairros e HOM2000.

4. No campo Taxa Multiplicativa de Correção, selecione 100000.

5. Execute o procedimento para criar a nova coluna, denominada GEBayes. Se desejar, altere seu nome no campo Nome da Nova Coluna.

Criando mapas temáticos a partir das taxas

Finalmente, vamos desenhar os mapas baseados nas taxas brutas e depois nas taxas Bayesianas. Podemos comparar estes dois mapas e verificar as diferenças entre eles. Com o tema ativo (Bairros) use a opção de Editar Legenda do menu popup, para criar um mapa temático de Quantil (4 fatias).
