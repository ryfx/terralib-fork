TerraView - Salvar Tema para Arquivo

O TerraView permite exportar temas de um banco de dados para os formatos shapefile, Spring e MID/MIF.
  • Shapefile: O formato shapefile armazena a informação geograficamente referenciada em vários arquivos distintos. O arquivo shp contém as geometrias, o arquivo dbf contém os atributos e o arquivo shx armazena a ligação entre os atributos e as geometrias.
  • Spring: conjunto de arquivos codificados em ASCII, onde cada arquivo armazena um tipo de entidade.
  • Mid/Mif: antes da grande difusão do SHP, era o formato utilizado para a importação e exportação de dados cartográficos associados a dados alfanuméricos. O arquivo mif contém as geometrias, e o arquivo mid contém os atributos
    1. Clique com o botão direito do mouse no tema que deseja exportar e selecione Exportar Tema para Arquivo.
    2. Selecione qual operação será executada, se deseja criar um arquivo a partir da colação total, ou se objetos consultados, etc...
    3. Selecione o formato de exportação, como descrito, podendo ser shapefile, Spring, MID/MIF e CSV (representação de ponto)
    4. Clique no botão Arquivo e defina o caminho onde o arquivo será salvo.
    5. Clique no botão OK para executar processo.
