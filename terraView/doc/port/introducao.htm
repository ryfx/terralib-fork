Introdução

A Figura abaixo representa a janela inicial do software TerraView no momento em que você inicia o seu uso.

TerraView é um visualizador de dados geográficos que inclui também algumas ferramentas de análise. Cada sessão TerraView trabalha com um banco de dados que contém os dados geográficos.
