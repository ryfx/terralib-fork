Importando Dados Geográficos

 

1. Depois de criar um banco TerraView, você poderá importar dados geográficos, constituídos de mapas e tabelas. Na barra de ferramentas, selecione o atalho representado pelo ícone para importar dados geográficos. Você pode realizar a mesma operação ativando o comando do menu Arquivo > Importar Dados. A janela Importação será aberta

2. Escolha a opção Arquivo para acessar o arquivo onde estão os dados que você quer importar. Uma janela para navegação nos diretórios de sua máquina será aberta.

3. Escolha o arquivo a ser importado. É possível importar dados nos formatos do ArcView (Shape files), do MapInfo (MID files) e do SPRING (SPRING-Geo files). Vá até o diretório Dados, selecione o tipo Shape file (*.shp) e selecione o arquivo Bairros.shp . Pressione o botão Abrir.

4. Você vai perceber que o TerraView volta à janela Importação, para que seja definida a projeção cartográfica dos dados importados. Pressione o botão Projeção.

5. A janela Projeção, onde serão definidos a Projeção, o Datum e os Parâmetros geográficos dos dados geográficos importados. Entre com os valores UTM (Universal Transverso de Mercator), SAD69, Zona 23 e pressione Executar.

6. Selecione a coluna de ligação com atributos. Essa coluna pode ser gerada automaticamente pelo TerraView. Neste caso, você pode deixar que o TerraView, gere automaticamente uma coluna de ligação.

Nota: O campo Encoding, quando habilitado, permite que seja especificada uma codificação de caracteres no plano a ser criado no banco de dados, que é diferente da codificação padrão (a qual depende do ambiente onde o servidor de banco de dados foi instalado). Este campo só será habilitado no caso de criação do banco de dados, mas apenas para alguns gerenciadores, como p.ex. PostgreSQL e PostGIS.

7. O sistema volta para janela Importação. Pressione o botão Executar para que os dados sejam importados para o banco de dados, criando uma novo Plano de Informação, p.ex.: Bairros. O sistema perguntará se você deseja a criação automática de identificador, numa janela de diálogo. Responda Sim.

O TerraView também perguntará se você deseja visualizar os dados importados:

Se você responder que não, o Plano de Informação será criado e inserida na Árvore de Bancos, mas ainda não poderá ser visualizada. Responda Sim. O TerraView cria automaticamente uma nova Vista "VistaBairros" e um novo Tema "Tema_Bairros". Dessa forma, você poderá visualizar seus dados na Tela de Visualização. Caso para visualizar os dados seja "não", a Vista será criada mas não mostrará o tema, para adicioná-lo deverá clicar no bota Tema e solicitar que seja adicionado aquela Vista.
