Fazendo Histogramas

 

Abra o banco de dados BH.mdb. Desative todos os temas da vista BH, exceto SetoresCensitarios. Com este tema selecionado, pressione o botão Desenhar . Como você poderá perceber, os polígonos do mapa estão transparentes; isso impossibilitará que se visualizem os apontamentos que você for realizar. Para alterar essa transparência, use o menu popup do tema SetoresCensitarios e escolha Visual > Default; escolha Estilo como Sólido e pressione Executar.

Na janela de grade no canto inferior da sua tela, você poderá perceber que existe, dentre outras, uma coluna REN_MED, contendo informações sobre a renda média mensal dos responsáveis pelo domicílio em cada setor censitário de Belo Horizonte. Você poderá obter a distribuição da renda média dos setores censitários para a cidade de Belo Horizonte.

1. Com o tema SetoresCensitarios ativo, use o menu popup e escolha Parâmetros do Gráfico.

2. A janela Parâmetros do Gráfico será aberta. Escolha o Tipo de Gráfico como Histograma. Com isso você irá obter a distribuição do atributo que você selecionar.

3. Escolha como Atributo(X) a coluna REN_MED (um atalho é selecionar o campo de Atributo(X) e pressionar a letra inicial R).

4. No campo Número de Fatias, você irá escolher em quantos intervalos você quer quebrar o gráfico. Deixe a opção padrão 30.

5. No campo Objetos você pode definir se quer fazer o histograma sobre todos os objetos, sobre os apontados, consultados e outros. Deixe a opção padrão Todos para fazer o histograma sobre todos os setores censitários de Belo Horizonte. Pressione Executar.

Na Tela de Visualização, o histograma mostra como os setores censitários em Belo Horizonte se distribuem segundo a renda média:

Ao analisá-lo, você nota que a grande parte dos setores censitários de BH tem renda média abaixo de R$ 1.000,00. Você pode querer, agora, analisar os outros dados ou as posições no mapa de menor renda média mensal.
