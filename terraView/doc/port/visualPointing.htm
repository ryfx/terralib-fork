Visual de Apontamento

O TerraView permite apontar para o mapa e descobrir seus atributos no grid, ou apontar no grid e descobrir sua localização no mapa.

A cor do visual de apontamento é definida para diferenciar visualmente os objetos apontados.

Note que apenas a cor é definida. Outros atributos de desenho como tipos de preenchimento de polígonos, tipos de traçado de linhas, tipos e tamanhos de pontos são mantidos no desenho do mapa. A diferenciação visual é feita apenas pela cor.

A cor do visual também é utilizada no grid para exibir os objetos apotados no mapa. As linhas correspondentes são pintadas com essa cor.

A cor do visual de apontamento é definida pela interface de seleção de cor.
