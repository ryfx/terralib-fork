Criando um Banco de Dados

 

Selecione a opção banco de dados a partir do atalho , ou pelo menu Arquivo > Banco de Dados:

1. Escolha a opção Criar.

2. Defina o tipo de banco de dados a ser criado. O TerraView pode utilizar vários SGBDs, como Access, Oracle ou MySQL. Selecione a opção Access.

3. Indique o diretório Bancos onde este novo banco de dados será salvo.

4. Escolha um nome para este banco de dados.

Nota: O campo Encoding, quando habilitado, permite que seja especificada uma codificação de caracteres no banco de dados a ser criado que é diferente da codificação padrão (a qual depende do ambiente onde o servidor de banco de dados foi instalado). Este campo só será habilitado no caso de criação do banco de dados, mas apenas para alguns gerenciadores, como p.ex. PostgreSQL e PostGIS.

6. Escolha o comando Executar para que os procedimentos anteriores sejam efetuados.

No caso de escolha do gerenciador ACCESS o banco será um arquivo com a extensão ".mdb", no diretório escolhido. No caso de outros gerenciadores o banco será criado no servidor escolhido. Nesse banco, você colocará os dados geográficos relativos à sua aplicação.
