Tamanho da Área de Desenho

O usuário poderá alterar o tamanho para a área de desenho.

  1. Para utilizar um tamanho diferente do tamanho default desta área, desmarque a seleção Tamanho Default da Janela.
  2. Selecione uma Largura e uma Altura entre as disponíveis ou informe outras arbitrárias.
  3. Clique no botão Executar.
