Renomeando e Removendo Camadas - Vistas e Temas

 

Quando o botão direito do mouse é pressionado sobre um Plano de Informação, Vista ou Tema, um menu pop-up é ativado para cada opção desejada (como mostrado abaixo, para uma Vista):


  • Renomear: você pode alterar o nome da camada, vista ou tema, colocando o nome que lhe é mais apropriado.
  • Remover: apagar definitivamente o Plano de Informação, Vista ou Tema do Banco de Dados.
  • Propriedades: esta opção fornece a você as propriedades do Plano de Informação, Vista ou Tema.

A opção Remover apaga definitivamente as informações do banco de dados caso seja na Árvore Banco de Dados - Plano de Informação. Quando um Plano de Informação é removido, serão automaticamente removidos todos os temas gerados a partir dele.

Quando uma vista é removida, todos os seus temas também serão. Esses temas poderão ser reconstruídos pois os dados encontram-se nos respectivos Planos de Informação.

Crie uma nova Vista chamada Crimes e adicione os temas Crimes2000, Crimes 2001, Crimes2002 e Crimes2003. Mova todos os temas da Vista Crimes para a Vista BH_Novo. Remova a vista vazia Crimes de sua árvore de vistas.

Remova a Vista BH_Novo a fim de obtermos o banco de dados original.
