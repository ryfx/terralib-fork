Importar Raster

 

Um arquivo raster é constituído por uma série de pontos, chamados "pixels" organizados em linhas e colunas. Um scanner captura uma imagem atribuindo uma linha, uma coluna e um valor de cor (branco e preto, escala de cinza ou cor) a cada ponto. Imagens de satélite e fotos aéreas são exemplos de dados raster.
O TerraView permite a importação dos seguintes tipos de arquivo: TIF, JPG, RAW e TXT. Para realizar a importação, siga os passos a seguir:

1. Clique no menu Arquivo - Importar Raster como mostra a Figura abaixo.


Acesso à importação raster através do Menu

2. Será aberta a janela inicial do Importar Raster como mostra Figura abaixo.


Janela inicial de importação raster

3. Usando o botão Arquivo indique onde a imagem está armazenada e clique no botão Abrir.

4. Assim que selecionar a imagem raster a ser importada, automaticamente aparecerão os números de Linhas e Colunas .


Interface após a escolha do arquivo com a imagem

O parâmetro Dummy é opcional e, se fornecido, indica o valor de pixel que será considerado como transparente, ou seja, não contém informação. Durante a importação, ele será mapeado no valor Dummy de saída, que indicará o valor de transparência na imagem armazenada no banco de dados. Os elementos da imagem com este valor não serão plotados na área de visualização.

5. Acione o botão Próximo para passar para a próxima janela, mostrada na Figura abaixo.


Janela de Características Geográficas

6. Nesta janela são mostradas as características geográficas da imagem obtidas a partir do arquivo de entrada, ou após eventual alteração da projeção, conforme explicação abaixo.

- A projeção informada às vezes pode não ser exatamente aquela com que a imagem se encontra no arquivo de entrada; procure informações sobre a projeção correta com o fornecedor da imagem.

7. Caso a projeção mostrada não esteja correta, clique no botão Projeção para alterá-la, informe os parâmetros corretos da projeção e clique no botão OK, voltando então à janela Características Geográficas;

8. Uma vez que a projeção esteja correta, clique no botão Próximo. Após isto será mostrada a janela da figura abaixo.


 Janela de Características de Armazenamento

9. Preencha os demais itens conforme se segue:

- Nome: no caso de um PI novo deve ser digitado, enquanto no caso de PI já existente deve ser selecionado a partir da lista fornecida de planos raster disponíveis. Este é o caso de criação de um mosaico, a partir do segundo arquivo;

- Object ID: que atualmente não deve ser preenchido;

- Compressão: a ser aplicada nos dados antes de serem armazenados, para diminuir o espaço físico ocupado; pode ser: nenhuma, zlib ou jpeg. A compressão jpeg é mais eficiente, porém não pode ser aplicada a imagens do tipo paleta;

- Altura e Largura, que definem o tamanho do bloco de dados usado no armazenamento; recomenda-se o valor 512 x 512;

- Dummy: o valor de transparência da imagem, conforme mencionado anteriormente. Este valor é sugerido na interface como sendo o mesmo valor usado no arquivo de entrada, informado na janela mostrada no passo 4 e explicado no texto que a ela se segue;

- Expansível: deve ser usada quando houver diversos arquivos de uma mesma imagem, ou seja, trata-se de um mosaico, e este é o primeiro arquivo a ser importado.

10. Pressione o botão Próximo , prosseguindo para o próximo passo que trata de multiresolução, ou então o botão Finalizar para iniciar a importação.

11. O passo opcional de multiresolução, tem como objetivo melhorar o desempenho na visualização da imagem, usando uma pirâmide de imagens obtidas da imagem original mas com resoluções diferentes, quanto mais próximo do topo menor a resolução. A imagem da forma como está no arquivo de entrada será sempre importada na base da pirâmide com a resolução original (nível 0).

Devem ser informados os seguintes parâmetros sobre os níveis adicionais de resolução que devem constar da pirâmide de multiresolução:

- Números de Níveis adicionais, com que a imagem será armazenada; este valor depende da aplicação e da escala de visualização que será utilizada normalmente, se for muito menor que a escala original da imagem, pode ser alto (p.ex. 10);

- Selecione os níveis adicionais na lista de níveis à direita: inicialmente a lista de níveis desejados para a pirâmide é mostrada com o número informado de níveis já selecionados, a partir do nível 1. Este é o caso recomendado, embora seja possível selecionar quaisquer níveis, mesmo que não sejam consecutivos, e em quantidade diferente do número de níveis originalmente informado.
