Exportação Vetorial

Até agora, você executou diversas formas de importação de dados para o TerraView . Esses dados podem ser exportados, no formato vetorial, ou seja, com informações geográficas em forma de vetores e com os atributos associados. Os formatos de exportação disponíveis são do SPRING, MID/MIF, Shapefile e TXT.

1. Escolha a camada a ser exportada. Selecione UP.

2. Escolha a tabela de atributos dessa camada a ser exportada. Como foi feita uma importação de tabela externa anteriormente, você poderá verificar que existem duas tabelas a serem exportadas, escolha apenas a tabela UP.

3. Escolha o formato do arquivo que será gerado. Selecione MID/MIF.

4. Defina o nome do arquivo exportado. Escolha o diretório Dados e o arquivo UP_2000.

5. Execute o procedimento.
