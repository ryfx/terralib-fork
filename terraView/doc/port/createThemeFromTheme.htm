Criar Tema a partir de Tema

Criando um tema a partir de outro tema é possível fazer uma cópia de um tema inteiro, ou de parte dele, conforme explicado abaixo.
Para criar um tema partindo de um tema, selecione o tema, clique com o botão direito do mouse abrindo o menu de contexto e selecione a opção Criar Tema a Partir de Tema, como mostra a figura a seguir.




A seguir, informe na área Seleção de Objetos qual subconjunto de objetos do tema será incluído no tema a ser criado. Várias opções estão disponíveis, conforme mostrado na figura abaixo.




Para incluir todos os objetos do tema no novo tema, use a opção Coleção.

Quando se deseja criar um tema com apenas uma parte dos objetos do tema original, deve-se ter previamente realizado alguma consulta por atributos ou consulta espacial, ou apontado para parte dos objetos, de forma que pelo menos uma das demais opções de seleção tenham efetiva utilidade.

Finalmente informe o nome do tema a ser criado e clique em Executar.
