Introdução ao Suporte WMS no TerraView

<< Anterior | Início | Próximo >>

Web Map Service ou, abreviadamente, WMS, é um tipo de serviço para produção de mapas na Web a partir de informações geo-referenciadas. O TerraView versão 3.2.0 oferece suporte para que você crie mapas (temas) provenientes de servidores WMS. A partir desta versão, o TerraView possui a habilidade de atuar como um cliente de serviços WMS, requisitando e apresentando mapas provenientes deste tipo de serviço.

Você poderá se conectar a servidores que atendam a qualquer uma das versões oficiais da especificação WMS fornecidas pelo consórcio Open Geospatial Consortium (OGC). Estas versões de servidores WMS são:

  • 1.3.0 de 15/03/2006, 03-109r1;
  • 1.1.1 de 16/01/2002, 01-068r3;
  • 1.1.0 de 21/01/2001, 01-047r2;
  • 1.0.0 de 19/04/2000, 00-028.

O plugin WMS Cliente incorpora ao TerraView as seguintes capacidades:

  • Criação de temas a partir de servidores WMS, para visualização integrada com os outros temas comuns do TerraView. Não há limites no número de servidores e mapas que podem ser acrescentados a uma mesma vista;

  • Exibição da legenda de um tema WMS, quando o servidor fornece adequadamente este suporte;

  • Associação de estilos no padrão SLD a temas WMS, quando o servidor fornece adequadamente este suporte;

  • Obtenção de informação sobre feições apresentadas em mapas provenientes de servidores WMS, através de apontamento sobre a área de desenho (visualização de mapas);

  • Obtenção de informação sobre os temas WMS: endereço do servidor, metadados fornecidos pelo servidor, informações de projeção e apresentação dos mapas.

Janela do TerraView apresentando mapas provenientes de serviços WMS

Na janela acima, os dois temas WMS são provenientes de um mesmo serviço WMS. No entanto, isto não é uma restrição, pois você poderá acrescentar em uma vista, mapas provenientes de serviços distintos. Basta abrir a janela de criação de temas WMS para acrescentar um novo mapa à janela do TerraView. Clique aqui para aprender a criar temas a partir de serviços WMS ou prossiga para a próxima página.

Mais Detalhes Sobre Serviços WMS:

As funcionalidades acima são possíveis porque um servidor WMS fornece um conjunto de operações padrões, que são:

  • GetCapabilities: retorna um documento XML, descrevendo quais as interfaces suportadas pelo servidor, quais as camadas de informação que podem ser servidas e em quais formatos de imagem ele pode servir essas camadas;

  • GetMap: permite que os clientes solicitem 'figuras' correspondentes aos mapas que podem ser servidos;

  • GetFeatureInfo: fornece maiores informações aos clientes a respeito de pontos específicos na imagem retornada pelo servidor.
