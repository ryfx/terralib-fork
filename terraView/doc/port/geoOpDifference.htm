Operação de Diferença entre Temas

A operação de diferença é feita entre temas com representação geométrica do tipo polígonos. É usada para remover objetos inteiros ou parte da geometria. No exemplo a seguir, o tema de entrada é o Tema1 que representa os municípios do Brasil. O tema com a área de recorte é o Tema2. A operação de diferença remove todos os municípios que estão totalmente contidos dentro da área elipsoidal do Tema2. Os municípios que estão parcialmente contidos não são removidos, entretanto, suas geometrias são aparadas (recortadas) pela elipse do Tema2. Interface de Diferença Entre Temas

A interface de diferença entre temas é mostrada nas Figuras 5, para acessá-la utilize o menu de contexto sobre a Vista, na opção Operações Geográficas - Diferença. Na caixa Tema 1 (Tema Original), seleciona-se o tema a ser recortado.


Figura 5 - Interface de intersecção entre temas - entrada vetorial.

Usando os botões na caixa de seleção do tema de entrada indique quais objetos devem ser recortados: todos, somente os apontados ou somente os selecionados pela consulta.

A caixa Tema 2 (Tema a ser subtraído do original), é usada para escolher qual tema será usado para servir de máscara de corte. Usando os botões, indique quais objetos devem ser usados como máscara: todos, somente os apontados ou somente os selecionados pela consulta.

Na caixa Nome do Plano de Saída entre com o nome desejado a fim de criar o plano de saída.

O botão Executar efetua a operação de diferença.

O botão Cancelar inibe a interface.

O botão Ajuda ativa a interface de ajuda.
