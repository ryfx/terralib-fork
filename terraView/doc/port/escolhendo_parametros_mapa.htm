Escolhendo os Parâmetros do Mapa

 

Você vai fazer um mapa onde, a partir do mínimo e do máximo de homicídios, serão definidos 5 intervalos de tamanho igual e a cada um será associada uma cor.

1. No campo Tipo escolha Passos Iguais, para criar intervalos de mesmo tamanho. Outros tipos de divisão de intervalos do atributo são:

  Tipo   Descrição
 Passos Iguais  Todos os grupos possuem o mesmo intervalo.  
 Quantil  Todos os grupos possuem aproximadamente o mesmo número de elementos.  
 Desvio Padrão  Todos os grupos possuem o mesmo intervalo, calculado como um múltiplo de 1/4, 1/2 or 1 desvio padrão  .
 Valor Único  Cada grupo corresponde a um valor distinto do atributo.  

2. No campo Número de Fatias, escolha opção 6, a fim de colorir o mapa com seis cores diferentes. Esta opção deve ser selecionada para os tipos Passos Iguais e Quantil. Para o tipo Valor Único o número de intervalos é definido automaticamente e para o tipo Desvio Padrão você deve selecionar entre intervalos de 0.25, 0.5 ou 1 desvio padrão.

3. No campo Atributo, escolha opção Bairros.POP2000. É neste campo que determinamos o atributo que será usado para colorir o mapa.

4. No campo Precisão, escolha opção 1. Aqui você determina o número de casas decimais que serão consideradas.

5. Clique em Agrupar. Os grupos foram criados, mas o mapa ainda não foi colorido.
