Visual Próprio

É possível alterar o visual de um determinado objeto do tema. Clique com o botão direito na Tela de Visualização sobre o objeto que queira mudar o visual. Um menu de contexto será apresentado. Clique em Visual Próprio - Definir o Visual Próprio do Objeto. Uma nova janela será exibida, como vemos na figura a seguir. As abas da janela do visual são habilitadas de acordo com as representações existentes no tema.

1. Altere o visual de acordo como o já explicado anteriormente no tópico Visual Padrão.

2. Clique em OK.

3. A Tela de Visualização é atualizada automaticamente exibindo o objeto com o novo visual.

Repita essa operação para os demais objetos que desejar alterar o visual. Para voltar a exibir os objetos com o visual padrão, clique em Visual Próprio - Remover o Visual Próprio do Objeto para remover o visual de um determinado objeto ou clique em Remover o Visual Próprio de Todos os Objetos no menu de contexto sobre a Tela de Visualização.
