Método da Soma

 

Esta operação adiciona vários temas para criar um novo plano de informação. Os temas devem ter o mesmo tipo de representação vetorial e devem ter apenas um tipo de geometria: somente polígonos, ou somente linhas, ou somente pontos ou somente células.

Nesta operação um tema é selecionado para servir de guia. O tema guia é usado como base. É a partir dele que se define o nome e o tipo dos atributos da nova tabela. Também é a partir dele que é definida a projeção das geometrias do novo plano de informação.

O Método da Soma permitirá a você unir dois mapas (ou temas) distintos transformando-o em outro mapa. Neste caso, iremos juntar as duas partes de Belo Horizonte para visualizar o mapa da cidade como um todo.

1. Escolha o tipo de representação gráfica;

2. Defina os temas de entrada, isto é, quais temas serão juntados;

3. Das tabelas (mapas) usados, defina os campos a serem mantidos;

4. Escolha um nome de saída para o novo mapa gerado;

5. Execute o procedimento.
