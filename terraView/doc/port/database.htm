Manipulação de Banco de Dados

A cada sessão de trabalho do TerraView é necessário haver pelo menos um Banco de Dados ativo. A Figura 1 mostra a interface que controla a manipulação de bancos de dados accessados pelo TerraView.

Criando um Banco de Dados
Figura 1 - Interface de manipulação de bancos de dados.
  1. Para criar um novo banco é necessário escolher a opção Criar na divisão Operação da interface Banco de Dados.
  2. No combo Tipo de Banco de Dados escolha o gerenciador que será usado para armazenar seu banco. Dependendo do gerenciador escolhido, a interface habilitará ou desabilitará outros itens (ver Figura 2):
    3. Figura 2 - Escolha do gerenciador.
  3. O gerenciador ACCESS exige que se escolha no botão Diretório onde o banco será criado.
  4. Os outros gerenciadores como o SQL Server, OracleSpatial, OracleAdo, MySQL, PostgreSQL e PostGIS, por serem servidores de bancos de dados, exigem que sejam preenchidos outros parâmetros (como mostra a Figura 3).
    5. Figura 3 - Parâmetros do servidor de banco de dados.
    Os campos a serem preenchidos nesse caso são:
    • Servidor: nome ou endereço da computador onde o servidor está instalado.
    • Porta: número da porta pela qual será feita a conexão.
    • Usuário: nome do usuário que acessa o servidor.
    • Senha: senha de acesso do usuário ao servidor.
  5. Defina o nome para o novo banco no campo Nome do Banco de Dados.
  6. O campo Encoding, quando habilitado, permite que seja especificada uma codificação de caracteres no banco de dados a ser criado que é diferente da codificação padrão (a qual depende do ambiente onde o servidor de banco de dados foi instalado). Este campo só será habilitado no caso de criação do banco de dados, mas apenas para alguns gerenciadores, como p.ex. PostgreSQL e PostGIS.
  7. Clique em Executar.
Após a sua criação, o banco de dados de dados é automaticamente conectado.

Abrindo uma conexão a um Banco de Dados previamente criado

  1. Para acessar um banco previamente criado é necessário escolher a opção Conectar na divisão Operação da interface Banco de Dados (ver Figura 4).
    Figura 4 - Conexão a um banco de dados previamente existente.

     

  2. No combo Tipo de Banco de Dados escolha o gerenciador usado para armazenar seu banco.
  3. Dependendo do gerenciador, ou tipo do banco de dados, preencha os outros campos da interface:
    • Gerenciador ACCESS: clique no botão Selecionar Banco de Dados e aponte para o arquivo .mdb que contém seu banco TerraLib. Veja que o nome do banco aparecerá no campo a frente.
    • Outros gerenciadores:
      • Preencha os campos Servidor, Porta, Usuário e Senha conforme explicado no item sobre a criação de um banco de dados novo.
      • Clique no botão Selecionar Banco de Dados. O combo à frente será preenchido com os bancos disponíveis nesse servidor.
      • Selecione nesse combo qual dos bancos disponíveis no servidor deseja se conectar.
  4. Clique em Executar.
OBS: se ao conectar o banco de dados o item Verificar Consistência no Banco de Dados estiver selecionado, o TerraView irá varrer o banco de dados e eliminar inconsistências e tabelas perdidas. Os bancos de dados conectados aparecem na área de banco de dados da interface principal como mostra a Figura 5.
