Visual de Consulta

A cor do visual de consulta é utilizada para diferenciar visualmente a execução de uma consulta no TerraView. A mesma cor é usada tanto na consulta espacial como na consulta por atributos.

Note que apenas a cor é definida. Outros atributos de desenho como tipos de preenchimento de polígonos, tipos de traçado de linhas, tipos e tamanhos de pontos são mantidos no desenho do mapa. A diferenciação visual é feita apenas pela cor.

A cor do visual também é utilizada no grid para exibir o resultado da consulta. As linhas que satisfazem são pintadas com essa cor.

A cor é definida pela interface de seleção de cor.

Uma outra forma para exibir o resultado de uma consulta é exibir tanto no mapa como no grid apenas os objetos que satisfazem a consulta. Essa é uma forma fácil e rápida de exibição, contudo, não é adotado pelo terraView porque assim perde o contexto na qual os objetos selecionados se encontram.

A forma adotada pelo TerraView, por manter o contexto, possibilita analisar melhor em torno dos objetos selecionados.
