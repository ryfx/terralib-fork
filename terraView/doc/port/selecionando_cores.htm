Selecionando Cores

 

As cores que aparecem na tela podem ser modificadas. Você pode criar sua própria mistura de cores de acordo com as opções de cores fornecidas pelo programa, utilizando os botões na parte inferior esquerda do Editor de Legenda. Cada botão possui uma letra que é referente à inicial da cor em inglês, como por exemplo, R (red) para vermelho e B (blue) para azul. Ao selecionar Mistura Gradual de Cores, você nota que a transição de uma cor para outra é mais suave. Você pode também escolher a variação do tom mais suave para o tom mais forte ou vice-versa e recomeçar sua seleção.

1. Clique em Executar para colorir o mapa de acordo com os grupos formados.

Faça agora outro mapa temático, com os mesmo parâmetros, substituindo apenas o tipo para Quantil e verifique a diferença. Para isso crie um novo tema chamado Bairros2. Você nota que, de acordo com os valores definidos para o intervalo dos atributos, a categorização dos dados é bastante diferente, podendo alterar o tipo de análise.
