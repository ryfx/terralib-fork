Animação

Essa ferramenta foi criada para animar o resultado de uma operação temporal ou de uma simulação.
Essa ferramenta é capaz de animar uma sequência de imagens com controle de velocidade, pausa e movimento passo a passo.
A ferramenta abre uma pasta e anima em sequência alfanumérica todas as imagems ali existentes.
Podemos citar como exemplos de tipos de arquivos aceitos: jpeg, png, bmp, etc.

O usuário deve criar uma pasta e inserir várias imagens sequenciais.

Clique com o botão direito do mouse na Tela de Visualização e selecione Animação, como mostra figura a seguir.



Clique no botão Diretório e selecione a pasta onde as imagens estão armazenadas.




Clique no botão Play para que as imagens sejam apresentadas.


Exemplo de aplicação: Temos um dado temporal que mostra a produção de leite das fazendas do Vale do Paraíba, podemos então fazer um agrupamento mensal com estes dados e, em seguida, fazer uma animação para ver como é a dinâmica da produção de leite dessas fazendas.
